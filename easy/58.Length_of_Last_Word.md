# 58. Length of Last Word

Link: https://leetcode.com/problems/length-of-last-word/
Tags: String
Difficulty: Easy

## 題目解釋
給你一個字串 `s`，由若干單詞組成，單詞前後用一些空格字符隔開。
返回字串中 **最後一個** 單詞的長度。
單詞是指僅由字母組成、不包含任何空格字符的最大子字串。

## 限制與條件
- `1 <= s.length <= 10^4`
- `s` 僅有英文字母和空格 `' '` 組成
- `s` 中至少存在一個單詞

## 解法思路
### 思考過程
題目要求最後一個單詞的長度。
代碼採用了 **從頭到尾遍歷** 的方式：
1. 維護一個計數器 `cnt`。
2. 當遇到非空格字符時，`cnt` 增加，並更新 `ans = cnt`（這樣 `ans` 始終是"當前正在計數的單詞長度"）。
3. 當遇到空格時，`cnt` 重置為 0。
這樣做的一個細節是：如果字串結尾有空格（例如 "Hello World "），在遇到最後的空格時 `cnt` 會歸零，但 `ans` 仍然保留著 "World" 的長度（5）。這剛好符合題目要求（忽略尾部空格，找最後一個單詞）。

### 解法：Forward Iteration
> [!IMPORTANT]
> 思路：遍歷統計連續字母長度，遇空格重置，始終記錄最近一次的單詞長度。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 遍歷字串一次。 |
| 空間複雜度 | O(1) | 僅使用常數空間。 |

```cpp
class Solution {
    public:
        int lengthOfLastWord(string s) {
            int cnt=0, ans=0;
            for(int i=0; i<s.length(); i++){
                if(s[i]==' '){
                    cnt=0;
                }
                else{
                    cnt++;
                    ans=cnt;
                }
            }
            return ans;
        }
    };
```
