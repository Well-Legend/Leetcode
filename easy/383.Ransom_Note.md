# 383. Ransom Note

**Link**: https://leetcode.com/problems/ransom-note/  
**Tags**: Hash Table, String, Counting  
**Difficulty**: Easy

## 題目解釋
給定兩個字串 `ransomNote` 與 `magazine`，判斷是否可以使用 `magazine` 裡的字元組成 `ransomNote`。  
`magazine` 中每個字元只能使用一次。

## 限制與條件
- `1 <= ransomNote.length, magazine.length <= 10^5`
- `ransomNote` 與 `magazine` 都只包含小寫英文字母

## 解法思路
### 思考過程
你的程式碼做法是字元計數差分：
1. 先遍歷 `ransomNote`，把每個字元需求次數加到 `cnt`。  
2. 再遍歷 `magazine`，把可提供的字元次數從 `cnt` 扣掉。  
3. 最後檢查 `cnt`：只要有任何字元的剩餘需求 `value > 0`，代表 `magazine` 不夠，回傳 `false`；否則回傳 `true`。  

這個方法直觀且效率高。

### 解法：Frequency Difference
> [!IMPORTANT]
> 思路：先加需求、再扣供給，最後看是否還有未滿足的字元需求。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n + m) | `n` 為 `ransomNote` 長度，`m` 為 `magazine` 長度。 |
| 空間複雜度 | O(k) | `k` 為字元種類數（英文小寫時可視為常數）。 |

```cpp
class Solution {
public:
    bool canConstruct(string ransomNote, string magazine) {
        unordered_map<char, int> cnt;
        for(char c:ransomNote){
            cnt[c]++;
        }
        for(char c:magazine){
            cnt[c]--;
        }
        for(auto &[c, value]: cnt){
            if(value>0){
                return false;
            }
        }
        return true;
    }
};
```
