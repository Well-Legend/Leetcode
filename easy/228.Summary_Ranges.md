# 228. Summary Ranges

**Link**: https://leetcode.com/problems/summary-ranges/  
**Tags**: Array  
**Difficulty**: Easy

## 題目解釋
給定一個已排序且不重複的整數陣列 `nums`，請把連續區間整理成字串：
- 若區間只有一個數字，直接輸出該數字
- 若區間包含多個連續數字，輸出 `"start->end"`

最後回傳所有區間字串組成的陣列。

## 限制與條件
- `0 <= nums.length <= 20`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `nums` 依遞增排序
- `nums` 中所有元素互不重複

## 解法思路
### 思考過程
你的程式碼用兩個指標 `left`、`right` 來表示目前區間的左右端點：
1. 一開始都指向同一個位置，代表目前先假設區間只有一個元素。  
2. 從第二個元素開始掃描，如果 `nums[i] == nums[i-1] + 1`，表示還在同一段連續區間內，就把 `right` 往右擴張。  
3. 一旦不連續，就把 `[left, right]` 這段輸出：
   - 若 `left == right`，加入單一數字
   - 否則加入 `"left->right"` 格式  
4. 接著把下一個位置設成新的區間起點，繼續往後掃。  
5. 迴圈結束後，別忘了把最後一段區間補進答案。  

整體就是線性掃描並把連續數字分組。

### 解法：Two Pointers
> [!IMPORTANT]
> 思路：用 `left`、`right` 維護一段連續區間，遇到斷點時就輸出當前區間。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 陣列只遍歷一次。 |
| 空間複雜度 | O(1) | 不計輸出結果，只使用常數變數。 |

```cpp
class Solution {
public:
    vector<string> summaryRanges(vector<int>& nums) {
        int left=0, right=0;
        vector<string> ans;
        if(nums.size()==0){
            return ans;
        }
        for(int i=1; i<nums.size(); i++){
            if (nums[i] == nums[i-1]+1){
                right++;
            }
            else{
                if(left==right){
                    ans.push_back(to_string(nums[left]));
                }
                else{
                    ans.push_back(to_string(nums[left]) + "->" + to_string(nums[right]));
                }
                right++;
                left=right;
                i=right;
            }
        }
        if(left==right){
            ans.push_back(to_string(nums[left]));
        }
        else{
            ans.push_back(to_string(nums[left]) + "->" + to_string(nums[right]));
        }
        return ans;
    }
};
```
