# 202. Happy Number

**Link**: https://leetcode.com/problems/happy-number/  
**Tags**: Hash Table, Math, Two Pointers  
**Difficulty**: Easy

## 題目解釋
給定一個正整數 `n`，反覆把它替換成「各位數字平方和」。  
如果最後可以得到 `1`，就稱它是快樂數（Happy Number），回傳 `true`；如果過程會陷入循環而無法得到 `1`，則回傳 `false`。

## 限制與條件
- `1 <= n <= 2^31 - 1`

## 解法思路
### 思考過程
你的程式碼使用 `unordered_set<int>` 來記錄已經出現過的中間結果：
1. 當前數字 `n` 不等於 `1` 時，計算它每一位數字平方後的總和 `keep`。  
2. 如果 `keep == 1`，代表成功到達快樂數，直接回傳 `true`。  
3. 如果 `keep` 還沒出現過，就放進集合 `happy`，繼續下一輪。  
4. 如果 `keep` 已經出現過，代表進入循環，不可能再到 `1`，回傳 `false`。  

這樣可以很直接地判斷是否重複進入同一個狀態。

### 解法：Hash Set Cycle Detection
> [!IMPORTANT]
> 思路：把每次轉換出的結果存進集合，若再次出現同一個值就代表陷入循環。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(k) | `k` 為轉換過程中的狀態數，實際上很小。 |
| 空間複雜度 | O(k) | 需要集合記錄已出現的中間值。 |

```cpp
class Solution {
public:
    bool isHappy(int n) {
        unordered_set<int> happy;
        while(n!=1){
            int digit=n;
            int keep=0;
            while(digit>0){
                int remain = digit%10;
                digit /= 10;
                keep += remain*remain; 
            }
            if(keep==1){
                return true;
            }
            if(happy.count(keep)==0){
                happy.insert(keep);
            }
            else{
                return false;
            }
            n=keep;
        }
        return true;
    }
};
```
