# 125. Valid Palindrome

**Link**: https://leetcode.com/problems/valid-palindrome/  
**Tags**: Two Pointers, String  
**Difficulty**: Easy

## 題目解釋
如果在將所有大寫字符轉換為小寫字符、並移除所有非字母數字字符之後，短語正著讀和反著讀都一樣，則可以認為該短語是一個回文。  
字母數字字符包括字母和數字。  
給你一個字串 `s` ，如果它是回文，返回 `true` ；否則，返回 `false` 。

## 限制與條件
- `1 <= s.length <= 2 * 10^5`  
- `s` 僅由可打印的 ASCII 字符組成

## 解法思路
### 思考過程
要判斷一個字串是否為回文，最簡單的方法是先對其進行處理，只保留字母和數字，並將其轉換為小寫。
1. 遍歷原始字串 `s`，檢查每個字元是否為字母或數字 (`isalnum`)。
2. 將符合條件的字元轉為小寫並存入一個新的字串 `alpha` 中。
3. 使用雙指針 `left` 和 `right` 分別指向 `alpha` 的頭部和尾部。
4. 比較兩側指標所指字元：
   - 若相同，則兩指標向中間移動 (`left++`, `right--`)。
   - 若不同，則直接返回 `false`。
5. 若指標交會後仍未發現不同，則返回 `true`。
6. 特別注意：空字串或處理後為空的字串應被視為回文。

### 解法：Pre-processing + Two Pointers
> [!IMPORTANT]
> 思路：先過濾非字母數字字元並轉化小寫，再以雙指針進行回文檢測。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | n 為字串長度。需要遍歷一遍字串進行處理，再遍歷一遍進行比對。 |
| 空間複雜度 | O(n) | 需要額外空間存儲處理後的字串。 |

```cpp
class Solution {
public:
    bool isPalindrome(string s) {
        string alpha;
        for (char letter : s){
            if(isalnum(letter)){
                alpha += tolower(letter);
            }
        }
        int n =alpha.size();
        int left=0, right=n-1;
        if (n==0){
            return true;
        }
        while (left<=right){
            if(alpha[left] == alpha[right]){
                left++;
                right--;
            }
            else{
                return false;
            }
        }
        return true;
    }
};
```
