# 1. Two Sum

**Link**: https://leetcode.com/problems/two-sum/  
**Tags**: Array, Hash Table  
**Difficulty**: Easy

## 題目解釋
給定一個整數陣列 `nums` 和一個整數目標值 `target`，請你在該陣列中找出和為目標值 `target` 的那兩個整數，並返回它們的陣列下標。  
你可以假設每種輸入只會對應一個答案。但是，陣列中同一個元素不能使用兩次。

## 限制與條件
- `2 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
- 只會存在一個有效答案

## 解法思路
### 思考過程
這是一個最基本的兩數之和問題。最直觀的暴力解法是檢查陣列中每一對元素的和是否等於目標值。  
我們可以使用兩層巢狀迴圈：
- 外層迴圈 `i` 從 0 到 `n-2`。
- 內層迴圈 `j` 從 `i+1` 到 `n-1`。
- 如果 `nums[i] + nums[j] == target`，則找到了這兩個數，返回 `[i, j]`。

### 解法：Brute Force
> [!IMPORTANT]
> 思路：遍歷所有可能的兩數組合，檢查其和是否為目標值。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n^2) | 兩層巢狀迴圈遍歷所有組合。 |
| 空間複雜度 | O(1) | 僅使用常數額外空間。 |

```cpp
class Solution 
{
public:
    vector<int> twoSum(vector<int>& nums, int target) 
    {
        vector<int> ans;
        int n=nums.size();
        for(int i=0;i<n-1;++i)
        {
            for(int j=i+1;j<n;++j)
            {
                if(nums[i]+nums[j]==target)
                {
                    ans.push_back(i);
                    ans.push_back(j);
                }
            }
        }
        return ans;
    }
};
```
