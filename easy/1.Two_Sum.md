# 1. Two Sum

**Link**: https://leetcode.com/problems/two-sum/  
**Tags**: Array, Hash Table  
**Difficulty**: Easy

## 題目解釋
給定整數陣列 `nums` 與整數 `target`，找出兩個不同索引 `i`、`j`，使得 `nums[i] + nums[j] == target`。  
回傳這兩個索引（順序不限），並保證每組輸入只有一個解，且同一個元素不能重複使用。

## 限制與條件
- `2 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
- 只會有一組有效答案

## 解法思路
### 思考過程
你的程式碼使用單次掃描 + `unordered_map`：
1. `map` 儲存「數值 -> 索引」。  
2. 掃描到 `nums[i]` 時，先算需要的補數 `sub = target - nums[i]`。  
3. 若 `sub` 已出現在 `map`，直接回傳 `{map[sub], i}`。  
4. 否則把目前數值與索引記進 `map`，繼續掃描。  

因為是先查補數再寫入自己，所以不會拿同一個元素重複配對。

### 解法：One-pass Hash Map
> [!IMPORTANT]
> 思路：邊走邊查補數，找到立即回傳。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個元素最多查詢/插入一次。 |
| 空間複雜度 | O(n) | 最多儲存 n 個數值索引。 |

```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int, int> map;
        for(int i=0; i<nums.size(); i++){
            int sub = target-nums[i];
            if(map.find(sub) != map.end()){
                return {map[sub], i};
            }
            map[nums[i]]=i;
        }
        return {};
    }
};
```
