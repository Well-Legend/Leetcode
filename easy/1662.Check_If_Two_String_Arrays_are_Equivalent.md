# 1662. Check If Two String Arrays are Equivalent

**Link**: https://leetcode.com/problems/check-if-two-string-arrays-are-equivalent/  
**Tags**: Array, String  
**Difficulty**: Easy

## 題目解釋
給你兩個字串陣列 `word1` 和 `word2`。如果兩個陣列表示的字串相同，返回 `true`；否則，返回 `false`。  
陣列表示的字串是由陣列中的所有元素按順序連接形成的字串。  

## 限制與條件
- `1 <= word1.length, word2.length <= 10^3`
- `1 <= word1[i].length, word2[i].length <= 10^3`
- `1 <= sum(word1[i].length), sum(word2[i].length) <= 10^3`
- `word1[i]` 和 `word2[i]` 由小寫字母組成

## 解法思路
### 思考過程
最直接的方法是將兩個陣列中的字串分別串接起來，形成兩個完整的字串，然後比較這兩個長字串是否相等。  
雖然也可以使用雙指針遍歷來節省空間，但直接串接的方法在資料規模較小時（如本題限制總長度 1000）非常直觀且易於實現。

### 解法：String Concatenation
> [!IMPORTANT]
> 思路：構建完整字串後直接比較。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(N + K) | N 為 word1 總字符數，K 為 word2 總字符數。需要遍歷所有字符進行串接和比較。 |
| 空間複雜度 | O(N + K) | 需要額外空間存儲串接後的兩個大字串。 |

```cpp
class Solution {
    public:
        bool arrayStringsAreEqual(vector<string>& word1, vector<string>& word2) {
            string str1="";
            string str2="";
            for(int i=0;i<word1.size();++i){
                str1+=word1[i];
            }
            for(int i=0;i<word2.size();++i){
                str2+=word2[i];
            }
            if (str1==str2)
                return true;
            else
                return false;
        }
};
```
