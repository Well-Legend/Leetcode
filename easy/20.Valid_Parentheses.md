# 20. Valid Parentheses

**Link**: https://leetcode.com/problems/valid-parentheses/  
**Tags**: String, Stack  
**Difficulty**: Easy

## 題目解釋
給定一個只包含括號字元的字串 `s`，判斷它是否有效。  
有效括號字串需要滿足：
- 左括號必須用相同類型的右括號閉合
- 左括號必須依照正確順序閉合

## 限制與條件
- `1 <= s.length <= 10^4`
- `s` 只包含 `()[]{}` 這六種字元

## 解法思路
### 思考過程
你的程式碼使用 `stack<char>`，而且不是把左括號壓入堆疊，而是直接把「期待的右括號」壓進去：
1. 遇到 `(` 時，把 `)` 推入堆疊。  
2. 遇到 `[` 時，把 `]` 推入堆疊。  
3. 遇到 `{` 時，把 `}` 推入堆疊。  
4. 遇到右括號時，檢查：
   - 堆疊是否為空
   - 堆疊頂端是否等於目前字元  
   若任一條件不符，直接回傳 `false`。  
5. 若匹配成功，就把堆疊頂端彈出。  

最後只要堆疊為空，代表所有括號都正確匹配。

### 解法：Stack of Expected Closings
> [!IMPORTANT]
> 思路：把預期出現的右括號放進堆疊，遇到實際右括號時直接比對。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個字元只進出堆疊一次。 |
| 空間複雜度 | O(n) | 最壞情況下堆疊會存下所有左括號。 |

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> stk;
for (char c:s) {
            if (c == '(') {
                stk.push(')');
            } else if (c == '[') {
                stk.push(']');
            } else if (c == '{') {
                stk.push('}');
            } 
            else {
                if (stk.empty() || stk.top() != c) {
                    return false; 
                }
                stk.pop();
            }
        }
        return stk.empty();
    }
};
```
