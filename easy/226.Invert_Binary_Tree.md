# 226. Invert Binary Tree

**Link**: https://leetcode.com/problems/invert-binary-tree/  
**Tags**: Tree, Depth-First Search, Breadth-First Search, Binary Tree  
**Difficulty**: Easy

## 題目解釋
給定一棵 binary tree 的根節點 `root`，請將整棵樹左右翻轉後回傳新的根節點。  
也就是對每個節點，交換它的左子樹與右子樹。

## 限制與條件
- 樹中節點數量在 `[0, 100]`
- `-100 <= Node.val <= 100`

## 解法思路
### 思考過程
你的程式碼使用遞迴 DFS：
1. 如果目前節點是 `nullptr`，直接回傳 `nullptr`。  
2. 否則先交換目前節點的 `left` 和 `right`。  
3. 交換完後，再遞迴處理新的左子樹與右子樹。  
4. 最後回傳原本的根節點 `root`。  

這樣每個節點都恰好完成一次左右交換。

### 解法：Recursive Swap
> [!IMPORTANT]
> 思路：對每個節點先交換左右子樹，再遞迴處理子樹。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點只處理一次。 |
| 空間複雜度 | O(h) | `h` 為樹高，來自遞迴堆疊。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(!root){
            return nullptr;
        }
        else{
            TreeNode* tmp = root->left;
            root->left = root->right;
            root->right = tmp;
        }
        invertTree(root->left);
        invertTree(root->right);
        return root;
    }
};
```
