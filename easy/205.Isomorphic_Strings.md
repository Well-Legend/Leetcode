# 205. Isomorphic Strings

**Link**: https://leetcode.com/problems/isomorphic-strings/  
**Tags**: Hash Table, String  
**Difficulty**: Easy

## 題目解釋
給定兩個字串 `s` 和 `t`，判斷它們是否同構（isomorphic）。  
同構代表 `s` 中每個字元都可以一對一映射到 `t` 中某個字元，且映射必須一致：
- 同一個字元不能映射到不同字元
- 不同字元不能映射到同一個字元

## 限制與條件
- `1 <= s.length <= 5 * 10^4`
- `t.length == s.length`
- `s` 與 `t` 由任意有效 ASCII 字元組成

## 解法思路
### 思考過程
你的程式碼使用兩個雜湊表做雙向檢查：
1. `mapst` 檢查 `s[i] -> t[i]` 的映射是否一致。  
2. `mapts` 檢查 `t[i] -> s[i]` 的反向映射是否一致。  

流程是先完整掃一次正向，再掃一次反向。  
只要任一方向出現不一致，就回傳 `false`；兩邊都一致才回傳 `true`。  
雙向驗證可以保證映射是「一對一」而非僅單向對應。

### 解法：Bidirectional Hash Mapping
> [!IMPORTANT]
> 思路：同時驗證 `s -> t` 與 `t -> s`，避免多對一或一對多映射。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 兩次線性掃描。 |
| 空間複雜度 | O(1) | ASCII 字元集大小固定（若視為一般字元集則為 O(k)）。 |

```cpp
class Solution {
public:
    bool isIsomorphic(string s, string t) {
        unordered_map<char, char> mapst, mapts;
        for(int i=0; i<s.size(); i++){
            if(mapst.find(s[i]) == mapst.end()){
                mapst[s[i]] = t[i];
            }
            if(mapst[s[i]] != t[i]){
                return false;
            }
        }
        for(int i=0; i<s.size(); i++){
            if(mapts.find(t[i]) == mapts.end()){
                mapts[t[i]] = s[i];
            }
            if(mapts[t[i]] != s[i]){
                return false;
            }
        }
        return true;
    }
};
```
