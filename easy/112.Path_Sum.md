# 112. Path Sum

**Link**: https://leetcode.com/problems/path-sum/  
**Tags**: Tree, Depth-First Search, Breadth-First Search, Binary Tree  
**Difficulty**: Easy

## 題目解釋
給定一棵 binary tree 的根節點 `root` 與整數 `targetSum`，判斷是否存在一條從根節點到葉節點的路徑，使得路徑上所有節點值總和等於 `targetSum`。

## 限制與條件
- 樹中節點數量在 `[0, 5000]`
- `-1000 <= Node.val <= 1000`
- `-1000 <= targetSum <= 1000`

## 解法思路
### 思考過程
你的程式碼使用 DFS 遞迴往下找：
1. 若目前節點是 `nullptr`，代表這條路徑不存在，回傳 `false`。  
2. 若目前節點是葉節點，就直接檢查 `targetSum == root->val`。  
3. 否則先把目前節點值從 `targetSum` 中扣掉，再遞迴往左子樹或右子樹繼續找。  

只要左右任一邊找到符合條件的根到葉路徑，就回傳 `true`。

### 解法：Recursive DFS
> [!IMPORTANT]
> 思路：一路扣掉目前節點值，走到葉節點時判斷剩餘值是否剛好等於該葉節點值。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點最多訪問一次。 |
| 空間複雜度 | O(h) | `h` 為樹高，來自遞迴堆疊。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    bool hasPathSum(TreeNode* root, int targetSum) {
        if(root == nullptr){
            return false;
        }
        if(!root->left && !root->right){
            return targetSum == root->val;
        }

        targetSum -= root->val;
        return hasPathSum(root->left, targetSum) || hasPathSum(root->right, targetSum);
    }
};
```
