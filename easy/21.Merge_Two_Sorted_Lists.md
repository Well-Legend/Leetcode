# 21. Merge Two Sorted Lists

**Link**: https://leetcode.com/problems/merge-two-sorted-lists/  
**Tags**: Linked List, Recursion  
**Difficulty**: Easy

## 題目解釋
給定兩個已排序的 linked list `list1` 與 `list2`，請將它們合併成一個新的已排序 linked list，並回傳合併後的頭節點。

## 限制與條件
- 兩個 linked list 的節點數量總和在 `[0, 50]`
- `-100 <= Node.val <= 100`
- `list1` 與 `list2` 都已按非遞減順序排序

## 解法思路
### 思考過程
你的程式碼建立一個 dummy 節點 `merge`，並用 `current` 指向目前合併串列的尾端。  

處理方式是：
1. 只要 `list1` 還沒走完，就持續比較 `list1` 和 `list2` 目前節點的值。  
2. 選出較小值後建立新節點接到結果串列尾端，並移動對應的 linked list 指標。  
3. 如果 `list2` 先空了，直接把 `list1` 剩下的部分接上。  
4. 主迴圈結束後，如果 `list2` 還有剩，就再把 `list2` 接到尾端。  

整體來說就是模擬兩條排序串列的 merge 流程。

### 解法：Iterative Merge
> [!IMPORTANT]
> 思路：用 dummy 節點串接答案，每次挑兩條串列目前較小的值接到尾端。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(m + n) | `m`、`n` 分別為兩條 linked list 長度。 |
| 空間複雜度 | O(m + n) | 這份寫法會為合併過程中新建立的節點配置額外空間。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {
        ListNode* merge = new ListNode();
        ListNode* current = merge;
        while(list1 != nullptr){
            if(list2 == nullptr){
                current->next = list1;
                break;
            }
            int val = min(list1->val, list2->val);
            if(list1->val > list2->val){
                list2 = list2->next;
            }
            else{
                list1 = list1->next;
            }
            current->next = new ListNode(val);
            current = current->next;
        }
        if(list2 != nullptr){
            while(current->next != nullptr){
                current = current->next;
            }
            current->next = list2;
        }
        return merge->next;
    }
};
```
