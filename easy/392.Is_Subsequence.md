# 392. Is Subsequence

**Link**: https://leetcode.com/problems/is-subsequence/  
**Tags**: Two Pointers, String, Dynamic Programming  
**Difficulty**: Easy

## 題目解釋
給定兩個字串 `s` 和 `t` ，判斷 `s` 是否為 `t` 的子序列。  
字串的一個子序列是透過刪除原字串中的某些（也可以不刪除）字符而不改變剩餘字符相對順序形成的新字串。  
（例如，`"ace"` 是 `"abcde"` 的一個子序列，而 `"aec"` 不是）。

## 限制與條件
- `0 <= s.length <= 100`
- `0 <= t.length <= 10^4`
- `s` 和 `t` 僅由小寫英文字母組成

## 解法思路
### 思考過程
這是一個典型的雙指針子序列匹配問題。
1. 我們可以用一個指針 `i` 來追蹤 `s` 中待匹配的字元位置。
2. 遍歷字串 `t` 中的每一個字元。
3. 如果 `t` 中的當前字元等於 `s[i]`，則代表找到了一個匹配，將 `i` 向後移一位。
4. 如果 `i` 到達了 `s` 的結尾，說明 `s` 的所有字元都已在 `t` 中按順序找到，返回 `true`。
5. 如果遍歷完 `t` 後 `i` 仍未到達 `s` 的結尾，返回 `false`。
6. 特別注意：如果 `s` 是空字串，它一定是任何字串的子序列，需返回 `true`。

### 解法：Two Pointers
> [!IMPORTANT]
> 思路：利用雙指針順序比對，當比對指針到達子字串末尾時即可判斷為子序列。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | n 為字串 `t` 的長度。最壞情況下需要遍歷整個 `t`。 |
| 空間複雜度 | O(1) | 僅使用常數額外空間。 |

```cpp
class Solution {
public:
    bool isSubsequence(string s, string t) {
        int i=0, n=s.size();
        if (n==0)
            return true;
        for(char letter : t){
            if (letter == s[i]){
                i++;
            }
            if(i == n){
                return true;
            }
        }
        return false;
    }
};
```
