# 3870. Count Commas in Range

**Link**: https://leetcode.com/problems/count-commas-in-range/  
**Tags**: Math  
**Difficulty**: Easy

## 題目解釋
給定整數 `n`，計算從 `1` 到 `n` 的所有整數在一般英文數字格式中，總共有多少個數字會出現逗號 `,`。  

以這份程式碼的邏輯來看，Easy 版本只會考慮「是否至少會出現一個逗號」，也就是數字是否達到四位數。

## 限制與條件
- 輸入為單一整數 `n`
- 本解使用 `int` 型別處理

## 解法思路
### 思考過程
你的程式碼抓住一個關鍵事實：  
只有從 `1000` 開始，數字格式中才會第一次出現逗號。  

因此：
1. 若 `n < 1000`，範圍內所有數字都沒有逗號，答案是 `0`。  
2. 若 `n >= 1000`，則從 `1000` 到 `n` 的每個數字都各自有一個逗號，所以答案就是這段區間的數量：`n - 999`。  

這讓整題可以直接用常數時間算出。

### 解法：Direct Math Counting
> [!IMPORTANT]
> 思路：四位數以上才會有逗號，因此只要計算 `[1000, n]` 有多少個數字。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(1) | 只做常數次判斷與計算。 |
| 空間複雜度 | O(1) | 不需要額外空間。 |

```cpp
class Solution {
public:
    int countCommas(int n) {
        if(n<1000){
            return 0;
        }
        else{
            return n-999;
        }
    }
};
```
