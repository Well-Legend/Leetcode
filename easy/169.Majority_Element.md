# 169. Majority Element

**Link**: https://leetcode.com/problems/majority-element/  
**Tags**: Array, Hash Table, Divide and Conquer, Sorting, Counting  
**Difficulty**: Easy

## 題目解釋
給定一個大小為 `n` 的陣列 `nums` ，返回其中的多數元素。多數元素是指在陣列中出現次數 **大於** `⌊ n/2 ⌋` 的元素。  
你可以假設陣列是非空的，並且給定的陣列總是存在多數元素。  

## 限制與條件
- `n == nums.length`
- `1 <= n <= 5 * 10^4`
- `-10^9 <= nums[i] <= 10^9`

## 解法思路
### 思考過程
題目保證存在一個出現次數超過 `n/2` 的多數元素。  
如果我們將陣列排序，那麼這個多數元素一定會覆蓋到陣列的中間位置（下標 `n/2`）。  
例如：  
- 如果多數元素最小，它會佔據 `0` 到 `>n/2` 的位置。  
- 如果多數元素最大，它會佔據 `<n/2` 到 `n-1` 的位置。  
- 無論如何，`nums[n/2]` 一定是那個多數元素。  

### 解法：Sorting
> [!IMPORTANT]
> 思路：排序後，中間的元素必為多數元素。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n log n) | 主要消耗在排序上。 |
| 空間複雜度 | O(log n) | 取決於排序算法的棧空間（C++ sort 通常是 IntroSort，空間為 log n）。 |

```cpp
class Solution {
    public:
        int majorityElement(vector<int>& nums) {
            sort(nums.begin(), nums.end());
            int ans = nums[nums.size()/2];
            return ans;
        }
    };
```
