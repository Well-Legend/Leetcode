# 242. Valid Anagram

**Link**: https://leetcode.com/problems/valid-anagram/  
**Tags**: Hash Table, String  
**Difficulty**: Easy

## 題目解釋
給定兩個字串 `s` 與 `t`，判斷 `t` 是否為 `s` 的字母異位詞（anagram）。  
也就是字元種類與每種字元出現次數都完全相同，只是排列順序可能不同。

## 限制與條件
- `1 <= s.length, t.length <= 5 * 10^4`
- `s` 與 `t` 只包含小寫英文字母

## 解法思路
### 思考過程
你的程式碼使用字元計數：
1. 先遍歷 `s`，把每個字元的出現次數記到 `match`。  
2. 若 `s`、`t` 長度不同，直接回傳 `false`。  
3. 再遍歷 `t`：  
   - 若該字元存在於 `match` 且計數還大於 0，就將計數減 1。  
   - 否則代表 `t` 出現了多餘字元或次數超過，直接回傳 `false`。  

若整輪都沒失敗，就代表 `t` 與 `s` 次數完全一致，回傳 `true`。

### 解法：Frequency Counting
> [!IMPORTANT]
> 思路：先累加 `s` 的字元次數，再用 `t` 去扣，過程中出現非法即失敗。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 兩次線性遍歷。 |
| 空間複雜度 | O(k) | `k` 為字元種類數（小寫字母時可視為常數）。 |

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        unordered_map<char, int> match;
        for(char alpha:s){
            if(match.find(alpha)==match.end()){
                match[alpha]=1;
            }
            else{
                match[alpha]++;
            }
        }
        if(s.size() != t.size()){
            return false;
        }
        for(char alpha:t){
            if(match.find(alpha)!=match.end() && match[alpha]>0){
                match[alpha]--;
            }
            else{
                return false;
            }
        }

        return true;
    }
};
```
