# 290. Word Pattern

**Link**: https://leetcode.com/problems/word-pattern/  
**Tags**: Hash Table, String  
**Difficulty**: Easy

## 題目解釋
給定字串 `pattern` 與字串 `s`，判斷 `s` 中的單字序列是否符合 `pattern`。  
`pattern` 的每個字元必須一對一對應到一個單字，且對應必須一致。

## 限制與條件
- `1 <= pattern.length <= 300`
- `1 <= s.length <= 3000`
- `pattern` 只包含小寫英文字母
- `s` 由小寫英文字母與空白組成

## 解法思路
### 思考過程
你的程式碼分兩步：
1. 先用 `stringstream` 把 `s` 切成 `words`。  
2. 用 `unordered_map<char, string>` 建立 `pattern[i] -> words[i]` 對應，並用 `unordered_set<string> seen` 防止兩個 pattern 字元映射到同一個單字。  

檢查流程：
- 若 `pattern` 長度和單字數不同，直接 `false`。  
- 若某 pattern 字元第一次出現：
  - 如果這個單字已被其他字元使用（`seen` 內有），回傳 `false`。  
  - 否則建立映射並加入 `seen`。  
- 若 pattern 字元已存在映射，必須和當前單字完全相同，否則 `false`。  

全部通過才回傳 `true`。

### 解法：Hash Map + Set (Bijection Check)
> [!IMPORTANT]
> 思路：用 map 檢查字元到單字一致性，用 set 保證不同字元不會共用同一單字。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 切字與遍歷 pattern/words 皆線性。 |
| 空間複雜度 | O(n) | 儲存單字陣列、映射與集合。 |

```cpp
class Solution {
public:
    bool wordPattern(string pattern, string s) {
        stringstream ss;
        unordered_map<char, string> map;
        unordered_set<string> seen;
        vector<string> words;
        string word;
        
        ss << s;
        while(ss >> word){
            words.push_back(word);
        }
        if(pattern.size() != words.size()){
            return false;
        }
        for(int i=0; i<pattern.size(); i++){
            if(map.find(pattern[i]) == map.end()){
                if(seen.count(words[i])){
                    return false;
                }
                map[pattern[i]] = words[i];
                seen.insert(words[i]);
            }
            else{
                if(!(map[pattern[i]] == words[i])){
                    return false;
                }
            }
        }
        return true;
    }
};
```
