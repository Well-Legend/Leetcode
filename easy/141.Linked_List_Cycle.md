# 141. Linked List Cycle

**Link**: https://leetcode.com/problems/linked-list-cycle/  
**Tags**: Hash Table, Linked List, Two Pointers  
**Difficulty**: Easy

## 題目解釋
給定一個 linked list 的頭節點 `head`，判斷鏈結串列中是否存在環。  
如果某個節點可以沿著 `next` 指標再次回到先前出現過的節點，就代表有 cycle。

## 限制與條件
- 節點數量範圍為 `[0, 10^4]`
- `-10^5 <= Node.val <= 10^5`
- `pos` 只作為測資描述，不會作為函式參數傳入

## 解法思路
### 思考過程
你的程式碼使用 Floyd Cycle Detection，也就是快慢指標：
1. `slow` 每次走一步，`fast` 每次走兩步。  
2. 如果 linked list 沒有環，`fast` 最後會先碰到 `nullptr`。  
3. 如果 linked list 有環，因為 `fast` 速度比較快，最終一定會在環內追上 `slow`。  
4. 因此只要在迴圈中出現 `fast == slow`，就可以直接回傳 `true`。  

這是判斷鏈結串列是否有環的經典做法，而且不需要額外記錄節點。

### 解法：Fast and Slow Pointers
> [!IMPORTANT]
> 思路：快指標一次走兩步、慢指標一次走一步；若存在環，快指標終會追上慢指標。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點最多被快慢指標有限次訪問。 |
| 空間複雜度 | O(1) | 僅使用兩個指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
        ListNode *fast = head;
        ListNode *slow = head;
        if(head == nullptr){
            return false;
        }
        while(fast != nullptr && fast->next != nullptr){
            fast = fast->next->next;
            slow = slow->next;
            if(fast==slow){
                return true;
            }
        }
        return false;
    }
};
```
