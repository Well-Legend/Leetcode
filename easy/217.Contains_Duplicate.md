# 217. Contains Duplicate

**Link**: https://leetcode.com/problems/contains-duplicate/  
**Tags**: Array, Hash Table, Sorting  
**Difficulty**: Easy

## 題目解釋
給定整數陣列 `nums`，判斷陣列中是否存在重複元素。  
如果任意數值至少出現兩次，回傳 `true`；否則回傳 `false`。

## 限制與條件
- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## 解法思路
### 思考過程
你的程式碼使用 `unordered_set<int>`：
1. 逐一遍歷 `nums`。  
2. 如果目前數字不在集合中，就插入集合。  
3. 如果已經存在，代表找到重複元素，立即回傳 `true`。  
4. 若整個陣列掃完都沒有重複，回傳 `false`。  

這是一個很直接的雜湊集合判重做法。

### 解法：Hash Set
> [!IMPORTANT]
> 思路：把看過的數字放進集合，只要再次遇到同一數字就直接返回。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個元素平均只做一次查詢與插入。 |
| 空間複雜度 | O(n) | 最多存下所有不同元素。 |

```cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> set;
        for(int num:nums){
            if(set.count(num) == 0){
                set.insert(num);
            }
            else{
                return true;
            }
        }
        return false;
    }
};
```
