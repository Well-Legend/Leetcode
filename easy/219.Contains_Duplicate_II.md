# 219. Contains Duplicate II

**Link**: https://leetcode.com/problems/contains-duplicate-ii/  
**Tags**: Array, Hash Table, Sliding Window  
**Difficulty**: Easy

## 題目解釋
給定整數陣列 `nums` 與整數 `k`，判斷是否存在兩個不同索引 `i`、`j`，使得：
- `nums[i] == nums[j]`
- `abs(i - j) <= k`

若存在則回傳 `true`，否則回傳 `false`。

## 限制與條件
- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `0 <= k <= 10^5`

## 解法思路
### 思考過程
你的程式碼使用 `unordered_map<int, int>` 記錄每個數字最近一次出現的位置：
1. 遍歷陣列 `nums`。  
2. 如果某個數字還沒出現過，就把它的索引記下來。  
3. 如果之前出現過，就計算這次索引和上次索引的差值。  
4. 若差值 `<= k`，表示找到符合條件的重複元素，直接回傳 `true`。  
5. 不管是否符合，都把該數字最後出現的位置更新為目前索引。  

這樣就只需要保留每個數字最新的位置即可。

### 解法：Hash Map of Last Seen Index
> [!IMPORTANT]
> 思路：追蹤每個數字最近一次出現的索引，遇到重複時檢查距離是否在 `k` 以內。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個元素只處理一次。 |
| 空間複雜度 | O(n) | 最多記錄所有不同元素的最後位置。 |

```cpp
class Solution {
public:
    bool containsNearbyDuplicate(vector<int>& nums, int k) {
        unordered_map<int, int> map;
        for(int i=0; i<nums.size(); i++){
            if(map.find(nums[i]) == map.end()){
                map[nums[i]] = i;
            }
            else{
                if(i-map[nums[i]] <= k){
                    return true;
                }
                map[nums[i]] = i;
            }
        }
        return false;
    }
};
```
