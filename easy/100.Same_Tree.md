# 100. Same Tree

**Link**: https://leetcode.com/problems/same-tree/  
**Tags**: Tree, Depth-First Search, Breadth-First Search, Binary Tree  
**Difficulty**: Easy

## 題目解釋
給定兩棵 binary tree 的根節點 `p` 和 `q`，判斷它們是否完全相同。  
完全相同代表：
- 結構一致
- 對應位置的節點值也一致

## 限制與條件
- 兩棵樹的節點數量都在 `[0, 100]`
- `-10^4 <= Node.val <= 10^4`

## 解法思路
### 思考過程
你的程式碼用遞迴來同時比對兩棵樹：
1. 如果 `p == q`，代表兩者同時是 `nullptr`，直接回傳 `true`。  
2. 如果其中一個是 `nullptr`、另一個不是，代表結構不同，回傳 `false`。  
3. 如果兩個節點值不同，也直接回傳 `false`。  
4. 否則繼續遞迴比對左右子樹。  

只要有任何一邊不同，最終結果就會是 `false`。

### 解法：Recursive Tree Comparison
> [!IMPORTANT]
> 思路：同步遞迴比對兩棵樹的節點值與左右子樹結構。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個對應節點最多比較一次。 |
| 空間複雜度 | O(h) | 來自遞迴堆疊，`h` 為樹高。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if(p==q){
            return true;
        }
        if(p==nullptr || q==nullptr){
            return false;
        }
        if(p->val != q->val){
            return false;
        }
        return isSameTree(p->left, q->left) && isSameTree(p->right, q->right);
    }
};
```
