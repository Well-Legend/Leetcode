## 檔案格式
每個解題檔案 (.md) 均包含以下結構：
- **題目資訊**：LeetCode 連結、標籤 (Tags)、難度。
- **題目解釋**：使用繁體中文詳細說明的題目要求。
- **限制與條件**：題目的輸入輸出限制。
- **解法思路**：
  - **思考過程**：一步步推導解法的邏輯。
  - **解法關鍵**：核心演算法或資料結構。
  - **複雜度分析**：時間與空間複雜度。
- **程式碼**：完整的 C++ 解答。
