# 76. Minimum Window Substring

**Link**: https://leetcode.com/problems/minimum-window-substring/  
**Tags**: Hash Table, String, Sliding Window  
**Difficulty**: Hard

## 題目解釋
給定兩個字串 `s` 與 `t`，請在 `s` 中找出一個最短子字串，使其包含 `t` 中所有字元（含重複次數）。  
若不存在，回傳空字串 `""`。

## 限制與條件
- `1 <= s.length, t.length <= 10^5`
- `s` 與 `t` 由英文字母組成

## 解法思路
### 思考過程
你的實作是滑動視窗 + 計數表：
1. 先用 `cnt` 建立 `t` 的需求次數，`need_cnt` 表示還缺多少個字元。  
2. `left` 先移到 `s` 中第一個出現在 `cnt` 的字元，避免前段無關字符。  
3. 右指針 `right` 擴張視窗：  
   - 若 `s[right]` 是需求字元，先看 `cnt[s[right]] > 0`，是的話就讓 `need_cnt--`。  
   - 接著 `cnt[s[right]]--`，表示視窗內已消化一個該字元。  
4. 當 `need_cnt == 0`，表示目前視窗已覆蓋 `t`：  
   - 嘗試更新最短答案。  
   - 再移動 `left` 收縮視窗，並回補 `cnt[s[left]]`；若因此讓某需求變成不足（`cnt[s[left]] > 0`），則 `need_cnt++`，停止收縮。  
   - 過程中也會跳過不在 `cnt` 的字元。  
5. 最後若有找到合法長度，就依 `start_idx` 與 `length` 組出答案字串。  

這種方式能在單次線性掃描中完成最短覆蓋視窗查找。

### 解法：Sliding Window + Frequency Map
> [!IMPORTANT]
> 思路：右邊擴張直到滿足需求，左邊收縮以求最短，透過 `need_cnt` 追蹤是否完整覆蓋。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(|s| + |t|) | 建表 O(|t|)，雙指針掃描 O(|s|)。 |
| 空間複雜度 | O(|Σt|) | 儲存 `t` 相關字元頻率。 |

```cpp
class Solution {
public:
    string minWindow(string s, string t) {
        int left=0, need_cnt=t.size();
        int start_idx, length=INT_MAX;
        string ans;
        unordered_map<char, int> cnt;
        for(int i=0; i<t.size(); i++){
            cnt[t[i]]=0;
        }
        for(int i=0; i<t.size(); i++){
            cnt[t[i]]++;
        }        

        while(left < s.size()){
            if(cnt.find(s[left]) != cnt.end()){
                break;
            }
            left++;
            if(left == s.size()){
                return ans;
            }
        }
        int right=left;
        while(right < s.size()){
            if(cnt.find(s[right]) != cnt.end()){
                if(cnt[s[right]] > 0){
                    need_cnt--;               
                }
                cnt[s[right]]--; 
            }
            while(need_cnt == 0){
                if((right-left+1)<length){
                    start_idx = left;
                    length = right-left+1;
                }
                cnt[s[left]]++;
                if (cnt[s[left]] > 0) {
                    need_cnt++;
                }
                left++;
                if(left>right){
                    break;
                }
                while(cnt.find(s[left]) == cnt.end()){
                    left++;
                }
            }
            right++;
        }
        if(length == INT_MAX){
            return ans;
        }
        for(int i=start_idx; i<(start_idx+length); i++){
            if(i==start_idx){
                ans = s[i];
            }
            else{
                ans += s[i];
            }
        }
        return ans;
    }
};
```
