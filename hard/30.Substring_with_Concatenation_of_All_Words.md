# 30. Substring with Concatenation of All Words

**Link**: https://leetcode.com/problems/substring-with-concatenation-of-all-words/  
**Tags**: Hash Table, String, Sliding Window  
**Difficulty**: Hard

## 題目解釋
給定字串 `s` 和字串陣列 `words`，其中 `words` 中每個字串長度相同。  
請找出所有起始索引，使得從該索引開始的子字串，剛好由 `words` 內所有字串串接而成（每個字要用一次，順序可不同，且中間不能有其他字元）。

## 限制與條件
- `1 <= s.length <= 10^4`
- `1 <= words.length <= 5000`
- `1 <= words[i].length <= 30`
- `words[i]` 長度相同

## 解法思路
### 思考過程
你的程式碼採用「分 offset 的固定長度滑動視窗」：
1. 先用 `map` 統計 `words` 每個單字需要的次數。  
2. 設 `word_len = words[0].size()`，然後針對 `offset = 0 ~ word_len - 1` 分開掃描，確保切詞邊界一致。  
3. 在每個 offset 中，指標 `idx` 每次跳 `word_len`，取出 `sub = s.substr(idx, word_len)`。  
4. 若 `sub` 在 `map` 中，就加入 `seen` 並增加 `seen_size`。  
   - 若某字超量（`seen[sub] > map[sub]`），就從左邊 `left` 逐字移除，直到不超量。  
5. 若 `sub` 不在 `map` 中，代表當前區段無效，清空 `seen`、重置 `seen_size`，並把 `left` 移到下一個位置。  
6. 當 `seen_size == words.size()`，表示視窗內剛好包含所需單字數量，記錄起點。  

透過 offset 分組，可以在 O(n) 的掃描中完成每組檢查。

### 解法：Sliding Window by Word Length
> [!IMPORTANT]
> 思路：以單字長度為步長做多組滑動視窗，維護視窗單字計數並在超量時收縮。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n * word_len)（常見寫法可視為 O(n)） | 每個 offset 做一次線性掃描，且每個單字進出視窗次數有限。 |
| 空間複雜度 | O(k) | `k` 為 `words` 中不同單字數，用於 `map` 與 `seen`。 |

```cpp
class Solution {
public:
    vector<int> findSubstring(string s, vector<string>& words) {
        unordered_map<string, int> map, seen;
        vector<int> ans;
        int word_len = words[0].size();
        if (s.empty() || words.empty()) return ans;

        for(string word:words){
            map[word]++;
        }
        for(int offset=0; offset<word_len; offset++){
            unordered_map<string, int> seen;
            int seen_size=0;
            int left = offset;
            for(int idx=offset; idx+word_len <= s.size(); idx+=word_len){
                string sub = s.substr(idx, word_len);
                if (map.find(sub) != map.end()){
                    seen[sub]++;
                    seen_size++;

                    while(seen[sub] > map[sub]){
                        string left_word = s.substr(left, word_len);
                        left += word_len;
                        seen[left_word]--;
                        seen_size--;
                    }
                }
                else{
                    seen.clear();
                    seen_size = 0;
                    left = idx + word_len;
                }


                if (seen_size == words.size()){
                    ans.push_back(idx - (seen_size-1)*word_len);
                }
            }
        }
        return ans;
    }
};
```
