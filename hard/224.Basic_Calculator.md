# 224. Basic Calculator

**Link**: https://leetcode.com/problems/basic-calculator/  
**Tags**: Math, String, Stack, Recursion  
**Difficulty**: Hard

## 題目解釋
給定一個字串 `s`，表示一個只包含非負整數、`+`、`-`、括號與空白的運算式。  
請計算並回傳這個運算式的值。

## 限制與條件
- `1 <= s.length <= 3 * 10^5`
- `s` 由數字、`+`、`-`、`(`、`)` 和空白組成
- 輸入保證是合法運算式

## 解法思路
### 思考過程
你的程式碼使用 stack 來處理括號前的計算狀態：
1. `curr` 用來累積目前正在讀的數字。  
2. `res` 記錄當前這一層括號內已計算好的結果。  
3. `sign` 表示接下來數字的正負號。  

掃描字串時：
- 遇到數字就持續累積到 `curr`。  
- 遇到 `+` 或 `-`，先把前一個數字 `sign * curr` 加進 `res`，再更新符號。  
- 遇到 `(`，把進括號前的 `res` 和 `sign` 推入 stack，然後重設當前層狀態。  
- 遇到 `)`，先把目前括號內最後一個數字結算進 `res`，再：
  1. 乘上括號前的符號
  2. 加回括號前已累積的結果  

最後掃描結束，再把剩下的 `curr` 納入答案。  
這樣就能正確處理巢狀括號與正負號。

### 解法：Stack for Parenthesized Expression State
> [!IMPORTANT]
> 思路：遇到左括號就保存上一層狀態，遇到右括號就把當前子表達式結果合併回上一層。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個字元只掃描一次。 |
| 空間複雜度 | O(n) | 最壞情況下 stack 會儲存多層括號狀態。 |

```cpp
class Solution {
public:
    int calculate(string s) {
        stack<int> st;
        long res = 0;
        long curr = 0;
        int sign = 1;

        for (char c : s) {
            if (isdigit(c)) {
                curr = curr * 10 + (c - '0');
            } else if (c == '+') {
                res += sign * curr;
                sign = 1;
                curr = 0;
            } else if (c == '-') {
                res += sign * curr;
                sign = -1;
                curr = 0;
            } else if (c == '(') {
                st.push(res);
                st.push(sign);
                res = 0;
                sign = 1;
            } else if (c == ')') {
                res += sign * curr;
                curr = 0;
                res *= st.top(); 
                st.pop();
                res += st.top(); 
                st.pop();
            }
        }

        res += sign * curr;
        return res;
    }
};
```
