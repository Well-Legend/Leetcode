# 73. Set Matrix Zeroes

**Link**: https://leetcode.com/problems/set-matrix-zeroes/  
**Tags**: Array, Matrix  
**Difficulty**: Medium

## 題目解釋
給定 `m x n` 矩陣 `matrix`，若某個元素為 `0`，則將它所在的整列與整行都設為 `0`。  
要求盡量使用原地修改，空間要優化。

## 限制與條件
- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

## 解法思路
### 思考過程
你的程式碼使用「第一列 + 第一欄作為標記區」：
1. 先掃描矩陣。遇到 `matrix[i][j] == 0`（`j >= 1`）時，將 `matrix[i][0]` 與 `matrix[0][j]` 標記為 `0`。  
2. 另外用 `first_row_zero` 記錄「第一欄是否本來就有 0」（變數名稱雖是 row，實際在追蹤第一欄）。  
3. 依照第一列標記，把對應欄位（從第 1 列開始）清零。  
4. 依照第一欄標記，把對應列（從第 1 欄開始）清零。  
5. 最後如果 `first_row_zero` 為真，把第一欄全部設為 `0`。  

這樣不用額外 `O(m+n)` 的標記陣列，只靠矩陣本身就能完成。

### 解法：In-place Marking
> [!IMPORTANT]
> 思路：利用第一列與第一欄保存「該列/欄是否要歸零」的資訊，最後統一套用。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(mn) | 需要多次線性掃描矩陣。 |
| 空間複雜度 | O(1) | 使用常數額外變數。 |

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        bool first_row_zero=false;
        int m=matrix.size(), n=matrix[0].size();

        for(int i=0; i<m; i++){
            if(matrix[i][0]==0){
                first_row_zero = true;
            } 
            for(int j=1; j<n; j++){
                if(matrix[i][j]==0){
                    matrix[i][0]=0;
                    matrix[0][j]=0;
                }
            }
        }

        for(int j=1; j<n; j++){
            if(matrix[0][j]==0){
                for(int i=1; i<m; i++){
                    matrix[i][j]=0;
                }
            }
        }
        for(int i=0; i<m; i++){
            if(matrix[i][0]==0){
                for(int j=1; j<n; j++){
                    matrix[i][j]=0;
                }
            }
        }

        if(first_row_zero){
            for(int i=0; i<m; i++){
                matrix[i][0]=0;
            }
        }
    }
};
```
