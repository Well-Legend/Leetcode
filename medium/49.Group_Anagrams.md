# 49. Group Anagrams

**Link**: https://leetcode.com/problems/group-anagrams/  
**Tags**: Array, Hash Table, String, Sorting  
**Difficulty**: Medium

## 題目解釋
給定字串陣列 `strs`，請把所有字母異位詞（anagrams）分組後回傳。  
字母異位詞指的是字元種類與次數相同，只是排列順序不同的字串。

## 限制與條件
- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 由小寫英文字母組成

## 解法思路
### 思考過程
你的程式碼使用「排序後的字串」作為分組鍵值：
1. 對每個字串 `str`，先複製成 `tmp`。  
2. 將 `tmp` 排序後，異位詞就會得到相同的排序結果。  
3. 用 `unordered_map<string, int>` 記錄這個排序字串對應到 `ans` 裡哪一組。  
4. 如果這個 key 第一次出現，就在 `ans` 新開一組並記錄索引。  
5. 如果 key 已存在，就把原字串加入對應分組。  

這樣就能把所有互為異位詞的字串集中到同一個陣列裡。

### 解法：Sorted String as Key
> [!IMPORTANT]
> 思路：把每個字串排序後當作 signature，相同 signature 的字串一定屬於同一組。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n * k log k) | `n` 為字串數量，`k` 為單一字串平均長度，主要成本來自排序。 |
| 空間複雜度 | O(nk) | 需要儲存分組結果與 hash key。 |

```cpp
class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, int> map;
        vector<vector<string>> ans;
        int idx=0;

        for(string str:strs){
            string tmp=str;
            sort(tmp.begin(), tmp.end());
            if(map.find(tmp) == map.end()){
                map[tmp] = idx;
                ans.push_back({str});
                idx++;
            }
            else{
                ans[map[tmp]].push_back(str);
            }
        }
        return ans;
    }
};
```
