# 289. Game of Life

**Link**: https://leetcode.com/problems/game-of-life/  
**Tags**: Array, Matrix, Simulation  
**Difficulty**: Medium

## 題目解釋
給定一個 `m x n` 的棋盤 `board`（`1` 代表活細胞、`0` 代表死細胞），依照康威生命遊戲規則更新到下一個狀態。  
更新必須同時發生，也就是不能讓先更新的格子影響到還沒更新的格子。

## 限制與條件
- `m == board.length`
- `n == board[i].length`
- `1 <= m, n <= 25`
- `board[i][j]` 為 `0` 或 `1`

## 解法思路
### 思考過程
你的程式碼用「狀態編碼」實現原地更新：
- `0`：死 -> 死（不變）
- `1`：活 -> 活（不變）
- `2`：死 -> 活（新生）
- `3`：活 -> 死（死亡）  

第一輪遍歷時：
1. 對每個格子計算 8 個鄰居中的活細胞數。  
2. 計算鄰居時，把 `1` 與 `3` 都視為「原本活著」，確保同步更新語意。  
3. 依規則決定是否標記成 `2` 或 `3`。  

第二輪遍歷時再把中間狀態收斂：
- `2 -> 1`
- `3 -> 0`

這樣就能在同一個矩陣中完成更新，不需額外複製棋盤。

### 解法：In-place State Encoding
> [!IMPORTANT]
> 思路：先用中間值記錄轉移，再第二次掃描還原成最終 0/1 狀態。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(mn) | 每格檢查固定 8 個鄰居，最後再掃一次收斂。 |
| 空間複雜度 | O(1) | 僅使用常數額外空間。 |

```cpp
class Solution {
public:
    void gameOfLife(vector<vector<int>>& board) {
        int m=board.size(), n=board[0].size();
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                int cnt=0;
                for(int hori_move=-1; hori_move<=1; hori_move++){
                    for(int ver_move=-1; ver_move<=1; ver_move++){
                        int ver_idx = i+ver_move;
                        int hori_idx = j+hori_move; 
                        if(ver_move==0 && hori_move==0){
                            continue;
                        }
                        if(ver_idx<0 || ver_idx>=m || hori_idx<0 || hori_idx>=n){
                            continue;
                        }
                        else{
                            int target = board[ver_idx][hori_idx];
                            if(target==1 || target==3){
                                cnt++;
                            }
                        }
                    }
                }
                if(board[i][j]==0){
                    if(cnt==3){
                        board[i][j]=2; // 0 to 1
                    }
                }
                else{
                    if(cnt<2 || cnt>3){
                        board[i][j]=3; // 1 to 0
                    }
                }
            }
        }

        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                if(board[i][j]==2){
                    board[i][j]=1;
                }
                else if(board[i][j]==3){
                    board[i][j]=0;
                }
            }
        }
    }
};
```
