# 92. Reverse Linked List II

**Link**: https://leetcode.com/problems/reverse-linked-list-ii/  
**Tags**: Linked List  
**Difficulty**: Medium

## 題目解釋
給定一個 linked list 的頭節點 `head`，以及兩個整數 `left`、`right`，請將位置 `left` 到 `right` 之間的節點反轉，並回傳新的頭節點。

## 限制與條件
- linked list 節點數量為 `n`
- `1 <= n <= 500`
- `-500 <= Node.val <= 500`
- `1 <= left <= right <= n`

## 解法思路
### 思考過程
你的程式碼先建立一個 `dummy` 節點，方便處理反轉區段剛好包含頭節點的情況。  

流程如下：
1. 先讓 `prev` 走到 `left` 前一個位置。  
2. `cur` 指向要反轉區段的第一個節點，`cur_after` 指向它下一個節點。  
3. 在 `[left, right]` 區間內逐步做指標反轉，讓後面的節點往前接回來。  
4. 反轉完成後，把：
   - 原本區段起點接到 `cur_after`
   - `prev` 接到新的區段頭 `cur`  

這樣就能只反轉中間一段，而不影響前後區段的連接。

### 解法：In-place Partial Reversal
> [!IMPORTANT]
> 思路：先走到反轉區段前一個節點，再對中間區段做原地指標反轉，最後重新接回前後兩端。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 最多走訪整條 linked list 一次。 |
| 空間複雜度 | O(1) | 僅使用常數額外指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* reverseBetween(ListNode* head, int left, int right) {
        ListNode* dummy = new ListNode(0, head);
        ListNode* prev = dummy;
        if(left==right){
            delete dummy;
            return head;
        }
        for(int i=1; i<left; i++){
            prev = prev->next;
        }
        ListNode* cur_prev = prev;
        ListNode* cur = cur_prev->next;
        ListNode* cur_after = cur->next;
        for(int i=left; i<right; i++){
            cur_prev = cur;
            cur = cur_after;
            cur_after = cur_after->next;
            cur->next = cur_prev;
        }
        prev->next->next = cur_after;
        prev->next = cur;
        return dummy->next;
    }
};
```
