# 82. Remove Duplicates from Sorted List II

**Link**: https://leetcode.com/problems/remove-duplicates-from-sorted-list-ii/  
**Tags**: Linked List  
**Difficulty**: Medium

## 題目解釋
給定一條已排序的 linked list，刪除所有出現重複的節點，只保留原本只出現一次的數字，並回傳處理後的 linked list。

## 限制與條件
- linked list 節點數量在 `[0, 300]`
- `-100 <= Node.val <= 100`
- linked list 已依非遞減順序排序

## 解法思路
### 思考過程
你的程式碼先建立一個 dummy 節點 `front` 指向原 linked list，並用 `clear` 保存結果起點。  

接著用 `end` 來掃描整條串列：
1. 如果 `end` 和 `end->next` 值相同，代表遇到重複區段。  
2. 記錄這個重複值 `tmp`，然後一路跳過所有相同值的節點。  
3. 如果當前節點沒有重複，就把它接到結果串列尾端，並讓 `front` 往前。  

最後再把結果尾端接到 `end`（通常是 `nullptr`），回傳 `clear->next`。

### 解法：Dummy Node + Skip Duplicate Block
> [!IMPORTANT]
> 思路：遇到重複值時整段跳過，只把真正只出現一次的節點接回答案串列。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點最多被掃描一次。 |
| 空間複雜度 | O(1) | 只使用常數額外指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode* front = new ListNode(0, head);
        ListNode* end = new ListNode();
        ListNode* clear = new ListNode();

        end = head;
        clear = front;
        while(end != nullptr){
            if(end->next != nullptr && end->val == end->next->val){
                int tmp = end->val;
                while(end != nullptr && end->val == tmp){
                    end = end->next;
                }
            }
            else{
                front->next = end;
                front = front->next;
                end = end->next;
            }
        }
        front->next = end;
        return clear->next;
    }
};
```
