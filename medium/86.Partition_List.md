# 86. Partition List

**Link**: https://leetcode.com/problems/partition-list/  
**Tags**: Linked List, Two Pointers  
**Difficulty**: Medium

## 題目解釋
給定一條 linked list 和整數 `x`，請將所有值小於 `x` 的節點移到值大於等於 `x` 的節點前面，並保留原本相對順序。

## 限制與條件
- linked list 節點數量在 `[0, 200]`
- `-100 <= Node.val <= 100`
- `-200 <= x <= 200`

## 解法思路
### 思考過程
你的程式碼建立兩條暫時的串列：
1. `small`：放所有 `< x` 的節點
2. `big`：放所有 `>= x` 的節點  

遍歷原 linked list 時：
- 若 `head->val < x`，就接到 `small` 串列尾端
- 否則接到 `big` 串列尾端  

最後把 `small` 的尾端接上 `bighead->next`，再把 `big->next` 設為 `nullptr`，避免尾端殘留舊連結。

這樣就能在保留相對順序的前提下完成分割。

### 解法：Two Dummy Lists
> [!IMPORTANT]
> 思路：分別維護小於 `x` 和大於等於 `x` 的兩條串列，最後再接起來。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點只遍歷一次。 |
| 空間複雜度 | O(1) | 只使用常數額外指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* partition(ListNode* head, int x) {
        ListNode* small = new ListNode();
        ListNode* smallhead = small;
        ListNode* big = new ListNode();
        ListNode* bighead = big;
        while(head != nullptr){
            if(head->val < x){
                small->next = head;
                small = small->next;
            }
            else{
                big->next = head;
                big = big->next;
            }
            head = head->next;
        }
        small->next = bighead->next;
        big->next = nullptr;
        return smallhead->next;
    }
};
```
