# 61. Rotate List

**Link**: https://leetcode.com/problems/rotate-list/  
**Tags**: Linked List, Two Pointers  
**Difficulty**: Medium

## 題目解釋
給定一條 linked list 的頭節點 `head` 和整數 `k`，將 linked list 向右旋轉 `k` 次後回傳新的頭節點。

## 限制與條件
- linked list 節點數量在 `[0, 500]`
- `-100 <= Node.val <= 100`
- `0 <= k <= 2 * 10^9`

## 解法思路
### 思考過程
你的程式碼先走訪 linked list，取得總長度 `n`，同時把尾節點連回頭節點，形成一個環。  

接著：
1. 把 `k` 取模成 `k % n`，避免多餘旋轉。  
2. 從原本頭節點開始往前走 `n - k` 步，找到新的頭節點位置。  
3. 用 `tail` 記錄新頭節點前一個節點，最後把 `tail->next = nullptr`，把環斷開。  

這樣就能完成旋轉，而不需要真的移動每個節點很多次。

### 解法：Make Cycle Then Cut
> [!IMPORTANT]
> 思路：先把 linked list 接成環，再走到新尾巴的位置斷開。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 掃描長度一次，找切點一次。 |
| 空間複雜度 | O(1) | 只使用常數額外指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* rotateRight(ListNode* head, int k) {
        int n=0;
        ListNode* allhead = head;
        if(head == nullptr || k==0){
            return head;
        }
        while(head != nullptr){
            if(head->next == nullptr){
                head->next = allhead;
                n++;
                break;
            }
            head = head->next;
            n++;
        }
        k = k%n;
        ListNode* tail = nullptr;
        for(int i=0; i<(n-k); i++){
            if(i == n-k-1){
                tail = allhead;
            }
            allhead = allhead->next;
        }
        tail->next = nullptr;
        return allhead;
    }
};
```
