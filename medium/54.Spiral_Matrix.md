# 54. Spiral Matrix

**Link**: https://leetcode.com/problems/spiral-matrix/  
**Tags**: Array, Matrix, Simulation  
**Difficulty**: Medium

## 題目解釋
給定一個 `m x n` 矩陣 `matrix`，請依照順時針螺旋順序回傳所有元素。

## 限制與條件
- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= m, n <= 10`
- `-100 <= matrix[i][j] <= 100`

## 解法思路
### 思考過程
你的程式碼是「方向移動 + 邊界收縮」的模擬法：  
1. 用四個邊界控制可走範圍：`upwall`、`downwall`、`leftwall`、`rightwall`。  
2. 以 `path` 表示目前方向（右、下、左、上）。  
3. 每次沿當前方向走到底，沿途把元素推入 `ans`，並減少 `remain`。  
4. 走完一段後，回退一格到合法位置、更新對應邊界，再切換到下一個方向。  
5. 當 `remain == 0` 時，代表所有元素都已輸出，結束迴圈。  

這種方式可以避免重複走訪，且每個元素只會被加入一次。

### 解法：Simulation (Boundary + Direction)
> [!IMPORTANT]
> 思路：沿四個方向循環前進，完成一段就收縮對應邊界，直到沒有剩餘元素。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(mn) | 每個元素恰好訪問一次。 |
| 空間複雜度 | O(1) | 不計輸出陣列 `ans`，僅使用常數變數。 |

```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        int upwall=0, downwall=matrix.size(), leftwall=-1, rightwall=matrix[0].size();
        int remain = matrix.size()*matrix[0].size();
        int col=0, row=0;
        int path=1; // 0左1右2上3下
        vector<int> ans;
        while(remain>0){
            switch(path){
                case 0:
                    while(col>leftwall){
                        ans.push_back(matrix[row][col]);
                        col--;
                        remain--;
                    }
                    col++;
                    row--;
                    leftwall++;
                    path=2;
                    break;

                case 1:
                    while(col<rightwall){       
                        ans.push_back(matrix[row][col]);
                        col++;
                        remain--;
                    }
                    col--;
                    row++;
                    rightwall--;
                    path=3;
                    break;

                case 2:
                    while(row>upwall){
                        ans.push_back(matrix[row][col]);
                        row--;
                        remain--;
                    }
                    row++;
                    col++;
                    upwall++;
                    path=1;
                    break;

                case 3:
                    while(row<downwall){
                        ans.push_back(matrix[row][col]);
                        row++;
                        remain--;
                    }
                    row--;
                    col--;
                    downwall--;
                    path=0;
                    break;
            }
        }
        return ans;
    }
};
```
