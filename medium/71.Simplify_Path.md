# 71. Simplify Path

**Link**: https://leetcode.com/problems/simplify-path/  
**Tags**: String, Stack  
**Difficulty**: Medium

## 題目解釋
給定一個 Unix-style 絕對路徑 `path`，請將它化簡成標準路徑。  
規則包括：
- 多個連續 `/` 視為單一 `/`
- `.` 代表目前目錄，可忽略
- `..` 代表回到上一層目錄
- 結果必須以單一 `/` 開頭，且不能有多餘尾斜線

## 限制與條件
- `1 <= path.length <= 3000`
- `path` 由英文字母、數字、`.`、`/`、`_` 組成
- `path` 是合法 Unix 路徑

## 解法思路
### 思考過程
你的程式碼用兩個 stack 來處理：
1. 先逐字掃描 `path`，用 `filename` 收集每一段目錄名稱。  
2. 每當遇到 `/`，就處理目前收集到的段落：
   - `"."`：忽略
   - `".."`：若 `tmpfile` 不空就彈出上一層
   - 其他字串：推入 `tmpfile`  
3. 字串掃描完後，還要再處理最後一段 `filename`。  
4. 因為 `tmpfile` 內的順序和輸出方向相反，你再用第二個 stack `truefile` 翻轉一次。  
5. 最後從 `truefile` 組回答案字串。若沒有任何有效目錄，回傳 `/`。  

這份寫法本質上就是模擬路徑進出資料夾的過程。

### 解法：Stack Simulation
> [!IMPORTANT]
> 思路：把有效目錄壓入堆疊，遇到 `..` 就彈出，最後再重建標準路徑。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個字元與每個目錄名稱最多處理常數次。 |
| 空間複雜度 | O(n) | 需要堆疊儲存路徑片段。 |

```cpp
class Solution {
public:
    string simplifyPath(string path) {
        stack<string> tmpfile, truefile;
        string filename;
        string ans;
        for(char c:path){
            if(c=='/'){
                if(filename.size()>0){
                    if(filename == "."){
                        filename.clear();
                        continue;
                    }
                    else if(filename == ".."){
                        if(!tmpfile.empty()){
                            tmpfile.pop();
                        }
                    }
                    else{
                        tmpfile.push(filename);
                    }
                    filename.clear();
                }
            }
            else{
                filename += c;
            }
        }
        if(filename.size()>0){
            if(filename == ".."){
                if(!tmpfile.empty()){
                    tmpfile.pop();
                }
            }
            else if(filename == "."){
                filename.clear();
            }
            else{
                tmpfile.push(filename);
            }
            filename.clear();
        }
        while(!tmpfile.empty()){
            truefile.push(tmpfile.top());
            tmpfile.pop();
        }
        if(truefile.empty()){
            return "/";
        }
        while(!truefile.empty()){
            ans += '/';
            filename = truefile.top();
            ans += filename;
            truefile.pop();
        }
        return ans;
    }
};
```
