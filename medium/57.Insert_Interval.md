# 57. Insert Interval

**Link**: https://leetcode.com/problems/insert-interval/  
**Tags**: Array  
**Difficulty**: Medium

## 題目解釋
給定一組已排序且互不重疊的區間 `intervals`，以及一個新區間 `newInterval`。  
請將 `newInterval` 插入到原本的區間陣列中，並在必要時合併重疊區間，最後回傳新的區間集合。

## 限制與條件
- `0 <= intervals.length <= 10^4`
- `intervals[i].length == 2`
- `0 <= start_i <= end_i <= 10^5`
- `newInterval.length == 2`
- `intervals` 依起點排序，且彼此不重疊

## 解法思路
### 思考過程
你的程式碼把整個流程分成三段：
1. 先把所有「完全在 `newInterval` 左邊，且不重疊」的區間直接放進答案。  
2. 接著處理所有和 `newInterval` 重疊的區間，持續更新 `newInterval` 的左右邊界，完成合併。  
3. 合併完成後，把新的 `newInterval` 放入答案，再把右邊剩下的區間接上。  

因為原本的 `intervals` 已經排序且互不重疊，所以只需要線性掃描一次，就能完成插入與合併。

### 解法：Three-phase Merge
> [!IMPORTANT]
> 思路：先收集左側不重疊區間，再合併中間重疊區間，最後接上右側區間。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個區間最多被掃描一次。 |
| 空間複雜度 | O(n) | 需要結果陣列儲存新區間集合。 |

```cpp
class Solution {
public:
    vector<vector<int>> insert(vector<vector<int>>& intervals, vector<int>& newInterval) {
        vector<vector<int>> ans;
        int i = 0;
        int n = intervals.size();
        
        while (i < n && intervals[i][1] < newInterval[0]) {
            ans.push_back(intervals[i]);
            i++;
        }
        while (i < n && intervals[i][0] <= newInterval[1]) {
            newInterval[0] = min(newInterval[0], intervals[i][0]);
            newInterval[1] = max(newInterval[1], intervals[i][1]);
            i++;
        }
        ans.push_back(newInterval);
        while (i < n) {
            ans.push_back(intervals[i]);
            i++;
        }
        return ans;
    }
};
```
