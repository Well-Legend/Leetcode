# 105. Construct Binary Tree from Preorder and Inorder Traversal

**Link**: https://leetcode.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/  
**Tags**: Array, Hash Table, Divide and Conquer, Tree, Binary Tree  
**Difficulty**: Medium

## 題目解釋
給定一棵 binary tree 的 preorder traversal 與 inorder traversal，請重建並回傳這棵樹的根節點。

## 限制與條件
- `1 <= preorder.length <= 3000`
- `inorder.length == preorder.length`
- `-3000 <= preorder[i], inorder[i] <= 3000`
- `preorder` 和 `inorder` 由唯一值組成

## 解法思路
### 思考過程
你的程式碼先用 `unordered_map<int, int>` 記錄每個值在 inorder 中的位置，讓切分區間時可以 `O(1)` 找到根節點索引。  

核心觀察：
- preorder 的當前位置一定是目前子樹的根
- inorder 中根節點左邊是左子樹，右邊是右子樹  

流程如下：
1. 用全域索引 `idx` 指向目前 preorder 要取的根節點。  
2. 建立根節點後，查出它在 inorder 的位置 `mid_idx`。  
3. 遞迴建立左子樹：`[start, mid_idx - 1]`  
4. 再遞迴建立右子樹：`[mid_idx + 1, end]`  

因為 preorder 是根左右，所以遞迴順序必須先左後右。

### 解法：Preorder + Inorder Recursive Reconstruction
> [!IMPORTANT]
> 思路：preorder 決定根節點順序，inorder 負責切出左右子樹範圍。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點建立一次，map 查詢為 O(1)。 |
| 空間複雜度 | O(n) | 需要 map 與遞迴堆疊。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    unordered_map<int,int> map;
    int idx=0;
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        for(int i=0; i<inorder.size(); i++){
            map[inorder[i]] = i;
        }        
        return build(preorder, 0, preorder.size()-1);
    }

    TreeNode* build(vector<int>& preorder, int start, int end){
        if(start > end){
            return nullptr;
        }
        TreeNode* root = new TreeNode(preorder[idx++]);
        int mid_idx = map[root->val];
        root->left = build(preorder, start, mid_idx-1);
        root->right = build(preorder, mid_idx+1, end);

        return root;
    }
};
```
