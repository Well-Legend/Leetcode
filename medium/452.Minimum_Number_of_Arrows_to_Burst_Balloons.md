# 452. Minimum Number of Arrows to Burst Balloons

**Link**: https://leetcode.com/problems/minimum-number-of-arrows-to-burst-balloons/  
**Tags**: Array, Greedy, Sorting  
**Difficulty**: Medium

## 題目解釋
有一些氣球以水平區間 `points[i] = [xstart, xend]` 表示，只要箭射在某個 `x` 位置，所有包含該位置的氣球都會被戳破。  
請回傳戳破所有氣球所需的最少箭數。

## 限制與條件
- `1 <= points.length <= 10^5`
- `points[i].length == 2`
- `-2^31 <= xstart < xend <= 2^31 - 1`

## 解法思路
### 思考過程
你的程式碼先把所有區間排序，然後用貪婪方式維護目前這支箭還能覆蓋的共同區間：
1. 如果 `points` 是空的，直接回傳 `0`。  
2. 排序後，先假設至少需要一支箭，並把 `current_end` 設成第一顆氣球的右端點。  
3. 依序看後面的氣球：
   - 若下一顆氣球的左端點 `<= current_end`，表示它和目前這支箭可覆蓋的範圍重疊，更新 `current_end = min(current_end, points[i][1])`，縮小共同交集。  
   - 若已經不重疊，代表必須再射一支新箭，並把 `current_end` 重設為新氣球的右端點。  

這樣每次都盡量讓同一支箭覆蓋最多重疊氣球，得到最少箭數。

### 解法：Greedy Interval Intersection
> [!IMPORTANT]
> 思路：排序後維護目前重疊區間的最右可行位置，遇到不重疊時才增加新箭。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n log n) | 主要成本為排序。 |
| 空間複雜度 | O(1) | 不計排序額外成本，只使用常數變數。 |

```cpp
class Solution {
public:
    int findMinArrowShots(vector<vector<int>>& points) {
        if (points.empty()) return 0;
        sort(points.begin(), points.end());
        
        int arrows = 1; 
        int current_end = points[0][1]; 
        
        for (int i = 1; i < points.size(); i++) {
            if (points[i][0] <= current_end) {
                current_end = min(current_end, points[i][1]);
            } 
            else {
                arrows++; 
                current_end = points[i][1]; 
            }
        }
        return arrows;
    }
};
```
