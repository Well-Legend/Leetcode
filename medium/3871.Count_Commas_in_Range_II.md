# 3871. Count Commas in Range II

**Link**: https://leetcode.com/problems/count-commas-in-range-ii/  
**Tags**: Math  
**Difficulty**: Medium

## 題目解釋
給定整數 `n`，計算從 `1` 到 `n` 的所有整數在一般英文數字格式中，總共會出現多少個逗號 `,`。  

和前一題不同的是，這一題需要考慮更大的數字，因此同一個數字可能會有不只一個逗號。

## 限制與條件
- 輸入為單一整數 `n`
- 本解使用 `long long` 型別處理

## 解法思路
### 思考過程
你的程式碼是按照「每跨過一個位數門檻，就多貢獻一層逗號」來累加：
- 從 `1000` 開始，每個數字至少有 `1` 個逗號，所以要加上 `n - 999`
- 從 `1,000,000` 開始，每個數字又多 `1` 個逗號，所以再加上 `n - 999999`
- 從 `1,000,000,000` 開始，再多加 `n - 999999999`
- 之後依此類推  

因此：
1. 先判斷 `n` 落在哪個區間。  
2. 把所有已達門檻的逗號層數貢獻累加進 `ans`。  
3. 最後回傳總數。  

這種寫法把原本逐一枚舉每個數字的做法，轉成依照數量區間直接計算。

### 解法：Piecewise Math Counting
> [!IMPORTANT]
> 思路：每超過一個千進位門檻，就把該門檻以上所有數字多出的那一層逗號數量一次加總。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(1) | 只有固定次數的區間判斷。 |
| 空間複雜度 | O(1) | 僅使用常數變數。 |

```cpp
class Solution {
public:
    long long countCommas(long long n) {
        long long ans=0;
        if(n<1000){
            ans=0;
        }
        else if(n >= 1000 && n<1000000){
            ans = (n - 999);
        }
        else if(n >= 1000000 && n<1000000000){
            ans = (n-999)+(n - 999999);
        }
        else if(n >= 1000000000 && n < 1000000000000LL){
            ans = (n - 999)+(n - 999999)+(n - 999999999);
            return ans;
        }
        else if(n >= 1000000000000LL && n < 1000000000000000LL){
            ans = (n - 999)+(n - 999999)+(n-999999999)+(n - 999999999999LL);
        }
        else{
            ans = (n - 999)+(n - 999999)+(n - 999999999)+(n - 999999999999LL)+(n - 999999999999999LL);
        }
        return ans;
    }
};
```
