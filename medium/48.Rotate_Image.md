# 48. Rotate Image

**Link**: https://leetcode.com/problems/rotate-image/  
**Tags**: Array, Math, Matrix  
**Difficulty**: Medium

## 題目解釋
給定一個 `n x n` 的二維矩陣 `matrix`，請將影像順時針旋轉 90 度。  
要求必須原地（in-place）修改，不可使用另一個矩陣來存放結果。

## 限制與條件
- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

## 解法思路
### 思考過程
你的程式碼使用經典的兩步驟：
1. 先做主對角線轉置（transpose）：`matrix[i][j]` 與 `matrix[j][i]` 交換。  
2. 再把每一列反轉（reverse row）。  

轉置會把列與行互換，接著每列反轉，就等同把整張圖順時針旋轉 90 度。

### 解法：Transpose + Reverse
> [!IMPORTANT]
> 思路：先沿主對角線交換，再反轉每一列，達成原地旋轉。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n^2) | 轉置與反轉總共遍歷矩陣量級。 |
| 空間複雜度 | O(1) | 只使用常數額外空間。 |

```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int row=matrix.size();
        int col=row;
        for(int i=0; i<row; i++){
            for(int j=i+1; j<col; j++){
                swap(matrix[i][j], matrix[j][i]);
            }
        }

        for(int i=0; i<row; i++){
            reverse(matrix[i].begin(), matrix[i].end());
        }
    }
};
```
