# 150. Evaluate Reverse Polish Notation

**Link**: https://leetcode.com/problems/evaluate-reverse-polish-notation/  
**Tags**: Array, Math, Stack  
**Difficulty**: Medium

## 題目解釋
給定一個字串陣列 `tokens`，表示一個 Reverse Polish Notation（後序表示法）的算式。  
請回傳這個算式的計算結果。  
可用運算子包含 `+`、`-`、`*`、`/`，其中除法需朝 0 截斷。

## 限制與條件
- `1 <= tokens.length <= 10^4`
- `tokens[i]` 是整數或四種運算子之一
- 輸入保證是一個合法的 RPN 算式

## 解法思路
### 思考過程
你的程式碼使用 `stack<int>` 來模擬 RPN 的運算過程：
1. 遇到數字時，直接轉成整數後推入堆疊。  
2. 遇到運算子時，先從堆疊彈出兩個數字：
   - 第二個彈出的值是左操作數 `a1`
   - 第一個彈出的值是右操作數 `a2`  
3. 根據運算子計算結果，再把結果推回堆疊。  
4. 最後堆疊頂端的值就是整個算式的答案。  

RPN 的特性正好非常適合用 stack 處理，因為每次運算都只需要最近的兩個操作數。

### 解法：Stack Simulation
> [!IMPORTANT]
> 思路：數字入堆疊，運算子就取出最近兩個數字計算，結果再放回堆疊。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個 token 只處理一次。 |
| 空間複雜度 | O(n) | 最壞情況下堆疊會存下多個中間結果。 |

```cpp
class Solution {
public:
    int evalRPN(vector<string>& tokens) {
        stack<int> stk;
        for(string c:tokens){
            if(c == "+"){
                int a2 = stk.top();
                stk.pop();
                int a1 = stk.top();
                stk.pop();
                a1 += a2;
                stk.push(a1);
            }
            else if(c == "-"){
                int a2 = stk.top();
                stk.pop();
                int a1 = stk.top();
                stk.pop();
                a1 -= a2;
                stk.push(a1);
            }
            else if(c == "*"){
                int a2 = stk.top();
                stk.pop();
                int a1 = stk.top();
                stk.pop();
                a1 *= a2;
                stk.push(a1);
            }
            else if(c == "/"){
                int a2 = stk.top();
                stk.pop();
                int a1 = stk.top();
                stk.pop();
                a1 /= a2;
                stk.push(a1);
            }
            else{
                int num = atoi(c.c_str());
                stk.push(num);
            }
        }
        return stk.top();
    }
};
```
