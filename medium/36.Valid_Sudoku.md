# 36. Valid Sudoku

**Link**: https://leetcode.com/problems/valid-sudoku/  
**Tags**: Array, Hash Table, Matrix  
**Difficulty**: Medium

## 題目解釋
給定一個 `9 x 9` 的數獨盤面 `board`，請判斷目前填入的數字是否有效。  
有效的條件是：
1. 每一列數字 `1~9` 不可重複。
2. 每一行數字 `1~9` 不可重複。
3. 每一個 `3 x 3` 子宮格中的數字 `1~9` 不可重複。  

注意：只需要驗證「目前已填入」的格子是否違規，`.` 代表空格可忽略，不需要判斷是否可解。

## 限制與條件
- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` 為 `'.'` 或 `1~9`

## 解法思路
### 思考過程
你的程式碼使用三個布林陣列來記錄是否出現過數字：
- `row[i][num]`：第 `i` 列是否出現過 `num`
- `col[j][num]`：第 `j` 行是否出現過 `num`
- `box[k][num]`：第 `k` 個 `3x3` 宮格是否出現過 `num`  

遍歷整個盤面時：
1. 若是 `'.'` 直接跳過。  
2. 把字元轉成索引 `num = board[i][j] - '1'`。  
3. 計算宮格編號 `k = (i / 3) * 3 + (j / 3)`。  
4. 只要 `row[i][num]`、`col[j][num]` 或 `box[k][num]` 有任一個已是 `true`，代表重複，立即回傳 `false`。  
5. 否則標記三個位置為 `true`，繼續檢查。  

若全部檢查完都沒有衝突，就回傳 `true`。

### 解法：Simulation + Boolean Table
> [!IMPORTANT]
> 思路：用三個 9x9 布林表，同步追蹤列、行、宮格是否出現過某數字。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(1) | 盤面固定為 81 格（可視為常數）。 |
| 空間複雜度 | O(1) | 使用固定大小的三個 `9x9` 陣列。 |

```cpp
class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        bool row[9][9] = {false};
        bool col[9][9] = {false};
        bool box[9][9] = {false};
        int num;
        for(int i=0; i<9; i++){
            for(int j=0; j<9; j++){
                if(board[i][j] == '.'){
                    continue;
                }
                num = board[i][j]-'1';
                int k = (i/3)*3 + (j/3);
                if(row[i][num] || col[j][num] || box[k][num]){
                    return false;
                }
                row[i][num] = true;
                col[j][num] = true;
                box[k][num] = true;
            }
        }
        return true;
    }
};
```
