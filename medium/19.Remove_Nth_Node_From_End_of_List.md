# 19. Remove Nth Node From End of List

**Link**: https://leetcode.com/problems/remove-nth-node-from-end-of-list/  
**Tags**: Linked List, Two Pointers  
**Difficulty**: Medium

## 題目解釋
給定一條 linked list 的頭節點 `head`，刪除倒數第 `n` 個節點，並回傳刪除後的頭節點。

## 限制與條件
- linked list 節點數量為 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

## 解法思路
### 思考過程
你的程式碼使用雙指標技巧，並先建立一個 dummy 節點 `removed` 指向原本的 `head`。  

流程是：
1. 讓 `end` 先往前走 `n` 步。  
2. 接著讓 `removed` 和 `end` 同步往前走，直到 `end->next == nullptr`。  
3. 這時 `removed` 剛好停在「要刪除節點的前一個位置」，直接把 `removed->next` 跳過即可。  

`front` 則用來保留 dummy 節點的起點，最後回傳 `front->next`。

### 解法：Two Pointers with Dummy Node
> [!IMPORTANT]
> 思路：先讓快指標領先 `n` 步，再同步前進，慢指標最後就會停在待刪節點前一格。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 最多遍歷 linked list 一次。 |
| 空間複雜度 | O(1) | 只使用常數額外指標。 |

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode* removed = new ListNode(0, head);
        ListNode* end = new ListNode();
        ListNode* front = new ListNode();
        front = removed;
        end = removed;
        int i=0;
        while(i<n){
            end = end->next;
            i++;
        }
        while(end->next != nullptr){
            removed = removed->next;
            end = end->next;
        }
        removed->next = removed->next->next;
        return front->next;
    }
};
```
