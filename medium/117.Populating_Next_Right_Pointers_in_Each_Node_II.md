# 117. Populating Next Right Pointers in Each Node II

**Link**: https://leetcode.com/problems/populating-next-right-pointers-in-each-node-ii/  
**Tags**: Linked List, Tree, Depth-First Search, Breadth-First Search, Binary Tree  
**Difficulty**: Medium

## 題目解釋
給定一棵 binary tree，請將每個節點的 `next` 指標指向同一層右邊的下一個節點。  
如果該節點已經是該層最右邊的節點，則將 `next` 設為 `nullptr`。

## 限制與條件
- 樹中節點數量在 `[0, 6000]`
- `-100 <= Node.val <= 100`

## 解法思路
### 思考過程
你的程式碼不用 queue，而是直接利用上一層已經建立好的 `next` 鏈來掃描每一層：
1. `curr` 指向目前正在處理的層。  
2. 每進入新的一層，就建立一個暫時的 dummy 節點 `child_head`，並用 `tail` 負責串接下一層的節點。  
3. 沿著目前這一層的 `next` 指標橫向走訪：
   - 若有 `left`，就接到下一層鏈上
   - 若有 `right`，也接到下一層鏈上  
4. 當這一層走完後，把 `curr` 移到 `child_head->next`，也就是下一層的最左節點。  

這樣就能一層一層地建立所有 `next` 關係。

### 解法：Level Traversal via Existing Next Pointers
> [!IMPORTANT]
> 思路：用當前層已經存在的 `next` 串列掃描，並動態串出下一層的 `next` 鏈。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點只會被處理一次。 |
| 空間複雜度 | O(1) | 不計遞迴與 queue，僅使用常數額外指標。 |

```cpp
/*
// Definition for a Node.
class Node {
public:
    int val;
    Node* left;
    Node* right;
    Node* next;

    Node() : val(0), left(NULL), right(NULL), next(NULL) {}

    Node(int _val) : val(_val), left(NULL), right(NULL), next(NULL) {}

    Node(int _val, Node* _left, Node* _right, Node* _next)
        : val(_val), left(_left), right(_right), next(_next) {}
};
*/

class Solution {
public:
    Node* connect(Node* root) {
        if(root == nullptr){
            return root;
        }

        Node* curr = root;
        while(curr != nullptr){
            Node* child_head = new Node();
            Node* tail = child_head;
            while(curr != nullptr){
                if(curr->left){
                    tail->next = curr->left;
                    tail = tail->next;
                }
                if(curr->right){
                    tail->next = curr->right;
                    tail = tail->next;
                }
                curr = curr->next;
            }
            curr = child_head->next;
        }
        return root;
    }
};
```
