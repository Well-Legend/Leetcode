# 106. Construct Binary Tree from Inorder and Postorder Traversal

**Link**: https://leetcode.com/problems/construct-binary-tree-from-inorder-and-postorder-traversal/  
**Tags**: Array, Hash Table, Divide and Conquer, Tree, Binary Tree  
**Difficulty**: Medium

## 題目解釋
給定一棵 binary tree 的 inorder traversal 與 postorder traversal，請重建並回傳這棵樹的根節點。

## 限制與條件
- `1 <= inorder.length <= 3000`
- `postorder.length == inorder.length`
- `-3000 <= inorder[i], postorder[i] <= 3000`
- `inorder` 和 `postorder` 由唯一值組成

## 解法思路
### 思考過程
你的程式碼同樣先用 `unordered_map<int, int>` 記錄每個值在 inorder 中的位置。  

這題的核心觀察是：
- postorder 的最後一個元素是目前子樹的根
- inorder 可以把根左右子樹的範圍切開  

流程如下：
1. 用 `idx` 從 postorder 最尾端開始往前取根節點。  
2. 建立根後，利用 inorder 的索引 `mid` 切出左右區間。  
3. 因為 postorder 是左、右、根，而我們現在從尾端往前讀，所以遞迴時要先建右子樹，再建左子樹。  

這樣才能正確對應 postorder 的讀取順序。

### 解法：Postorder + Inorder Recursive Reconstruction
> [!IMPORTANT]
> 思路：postorder 從尾端提供根節點，inorder 切出左右區間；遞迴順序必須先右後左。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點建立一次，map 查詢為 O(1)。 |
| 空間複雜度 | O(n) | 需要 map 與遞迴堆疊。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    unordered_map<int,int> map;
    TreeNode* buildTree(vector<int>& inorder, vector<int>& postorder) {
        int idx = postorder.size()-1;
        for(int i=0; i<postorder.size(); i++){
            map[inorder[i]] = i;
        }

        return build(postorder, 0, postorder.size()-1, idx);
    }

    TreeNode* build(vector<int>& postorder, int start, int end, int& idx){
        if(start>end){
            return nullptr;
        }
        TreeNode* root = new TreeNode(postorder[idx--]);
        int mid = map[root->val];
        root->right = build(postorder, mid+1, end, idx);
        root->left = build(postorder, start, mid-1, idx);

        return root;
    }
};
```
