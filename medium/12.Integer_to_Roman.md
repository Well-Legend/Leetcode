# 12. Integer to Roman

Link: https://leetcode.com/problems/integer-to-roman/
Tags: Hash Table, Math, String
Difficulty: Medium

## 題目解釋
羅馬數字包含以下七種字符： `I`, `V`, `X`, `L`, `C`, `D` 和 `M`。
給定一個整數，將其轉換為羅馬數字。

## 限制與條件
- `1 <= num <= 3999`

## 解法思路
### 思考過程
我們可以將整數按位處理（個位、十位、百位、千位）。
對於每一位，根據其數值（0-9），映射到對應的羅馬數字表示。
代碼中採用的方法是：
- 循環取出整數的每一位（`num % 10`），並使用一個變數 `round` 記錄當前是哪一位（0: 個位, 1: 十位, ...）。
- 使用 `switch` 或 `if-else` 硬編碼處理每一位的特殊規則（如 4 是 IV/XL/CD, 9 是 IX/XC/CM, 5 是 V/L/D 等）。
- 將生成的羅馬字符存入臨時陣列，最後反轉拼接。
另一種常見的做法是使用貪婪算法，建立一個數值到羅馬字符的映射表（1000->M, 900->CM, ...），從大到小儘量減去最大值。但本題代碼使用的是逐位映射法。

### 解法：Digit-by-Digit Mapping
> [!IMPORTANT]
> 思路：分解整數的每一位，根據位權與數值轉換為對應的羅馬字符組合。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(log n) | 或是 O(1)，因為輸入範圍限制在 3999，位數最多 4 位，循環次數固定。 |
| 空間複雜度 | O(1) | 僅使用常數空間。 |

```cpp
class Solution {
    public:
        string intToRoman(int num) {
            char tmp[100];
            string ans;
            int round=0, cnt=0;
            while(num!=0){
                int word=num%10;
                switch(round){
                    case 0:
                        while(word!=0){
                            if(word==9){
                                tmp[cnt]='X';
                                tmp[cnt+1]='I';
                                cnt+=2;
                                word-=9;
                            }
                            else if(word==4){
                                tmp[cnt]='V';
                                tmp[cnt+1]='I';
                                cnt+=2;
                                word-=4;
                            }
                            else if(word==5){
                                tmp[cnt++]='V';
                                word-=5;
                            }
                            else{
                                tmp[cnt++]='I';
                                word--;
                            }
                        }
                        break;
    
                    case 1:
                        while(word!=0){
                            if(word==9){
                                tmp[cnt]='C';
                                tmp[cnt+1]='X';
                                cnt+=2;
                                word-=9;
                            }
                            else if(word==4){
                                tmp[cnt]='L';
                                tmp[cnt+1]='X';
                                cnt+=2;
                                word-=4;
                            }
                            else if(word==5){
                                tmp[cnt++]='L';
                                word-=5;
                            }
                            else{
                                tmp[cnt++]='X';
                                word--;
                            }
                        }
                        break;
    
                    case 2:
                        while(word!=0){
                            if(word==9){
                                tmp[cnt]='M';
                                tmp[cnt+1]='C';
                                cnt+=2;
                                word-=9;
                            }
                            else if(word==4){
                                tmp[cnt]='D';
                                tmp[cnt+1]='C';
                                cnt+=2;
                                word-=4;
                            }
                            else if(word==5){
                                tmp[cnt++]='D';
                                word-=5;
                            }
                            else{
                                tmp[cnt++]='C';
                                word--;
                            }
                        }
                        break;
    
                    case 3:
                        while(word!=0){
                            tmp[cnt++]='M';
                            word--;
                        }
                        break;
                }
                num/=10;
                round++;
            }
    
            for(int i=cnt-1; i>=0; i--){
                ans+=tmp[i];
            }
            return ans;
        }
    };
```
