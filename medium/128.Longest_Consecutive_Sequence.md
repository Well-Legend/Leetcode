# 128. Longest Consecutive Sequence

**Link**: https://leetcode.com/problems/longest-consecutive-sequence/  
**Tags**: Array, Hash Table, Union Find  
**Difficulty**: Medium

## 題目解釋
給定一個未排序的整數陣列 `nums`，找出其中最長的連續整數序列長度。  
要求演算法時間複雜度為 `O(n)`。

## 限制與條件
- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## 解法思路
### 思考過程
你的程式碼先把所有數字放進 `unordered_set<int>`，讓查找某個數字是否存在可以在平均 `O(1)` 完成。  

接著遍歷集合中的每個數字 `num`：
1. 先檢查 `num - 1` 是否存在。  
2. 如果不存在，代表 `num` 是某段連續序列的起點。  
3. 從這個起點開始，用 `while(set.count(num + k) > 0)` 一路往右找，計算這段序列長度 `cnt`。  
4. 每找到一段，就更新最大長度。  

這個關鍵設計是「只從序列起點開始展開」，避免對同一段連續序列重複計算。

### 解法：Hash Set Expansion
> [!IMPORTANT]
> 思路：把所有數字放進集合，只從沒有前驅的數字開始往右延伸計算序列長度。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個數字最多被當作序列元素延伸一次。 |
| 空間複雜度 | O(n) | 需要集合儲存所有數字。 |

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int> set(nums.begin(), nums.end());
        int max=0;
        for(int num:set){
            if(set.count(num-1) == 0){
                int k=0, cnt=0;
                while(set.count(num+k) > 0){
                    k++;
                    cnt++;
                }
                if(cnt>max){
                    max = cnt;
                }
            }
        }
        return max;
    }
};
```
