# 129. Sum Root to Leaf Numbers

**Link**: https://leetcode.com/problems/sum-root-to-leaf-numbers/  
**Tags**: Tree, Depth-First Search, Binary Tree  
**Difficulty**: Medium

## 題目解釋
給定一棵 binary tree，每個節點都包含一個 `0-9` 的數字。  
每一條從根節點到葉節點的路徑都可以組成一個數字，請回傳所有這些數字的總和。

## 限制與條件
- 樹中節點數量在 `[1, 1000]`
- `0 <= Node.val <= 9`
- 樹的深度不超過 `10`

## 解法思路
### 思考過程
你的程式碼用 DFS 往下累積目前路徑代表的數字：
1. `num` 表示從根走到目前節點之前所形成的數字。  
2. 到達目前節點時，更新成 `num = 10 * num + node->val`。  
3. 若目前節點是葉節點，就直接回傳這個完整數字。  
4. 否則繼續往左子樹與右子樹遞迴，最後把兩邊結果加總。  

這樣每條根到葉的路徑都會剛好產生一個數字。

### 解法：DFS with Running Number
> [!IMPORTANT]
> 思路：沿路把數字左移一位再加上當前節點值，到葉節點時直接回傳該路徑數字。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 每個節點只會訪問一次。 |
| 空間複雜度 | O(h) | `h` 為樹高，來自遞迴堆疊。 |

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int sumNumbers(TreeNode* root) {
        return dfs(root, 0);
    }

    int dfs(TreeNode* node, int num){
        if(node == nullptr){
            return 0;
        }
        num = 10*num + node->val;
        if(!node->left && !node->right){
            return num;
        }
        return dfs(node->left, num) + dfs(node->right, num);
    }
};
```
