# 138. Copy List with Random Pointer

**Link**: https://leetcode.com/problems/copy-list-with-random-pointer/  
**Tags**: Hash Table, Linked List  
**Difficulty**: Medium

## 題目解釋
給定一條 linked list，其中每個節點除了 `next` 指標外，還有一個 `random` 指標，可以指向任意節點或 `nullptr`。  
請建立這條 linked list 的深拷貝（deep copy），並回傳新 linked list 的頭節點。

## 限制與條件
- `0 <= n <= 1000`
- `-10^4 <= Node.val <= 10^4`
- `random` 可以指向任意節點或 `nullptr`

## 解法思路
### 思考過程
你的程式碼使用 `unordered_map<Node*, Node*>` 來建立「原節點 -> 新節點」的對應關係。

流程分兩輪：
1. 第一輪遍歷原 linked list，先把每個原節點對應的新節點建立出來，只複製 `val`。  
2. 額外設定 `map[nullptr] = nullptr`，這樣後面處理 `next` 和 `random` 時就能直接查表。  
3. 第二輪再遍歷一次，把每個新節點的：
   - `next`
   - `random`  
   都透過 map 連到對應的新節點。  

最後回傳 `map[head]`，也就是新 linked list 的頭。

### 解法：Hash Map Node Mapping
> [!IMPORTANT]
> 思路：先複製所有節點，再利用原節點到新節點的映射補齊 `next` 和 `random`。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n) | 兩次遍歷 linked list。 |
| 空間複雜度 | O(n) | 需要 hash map 記錄原節點與新節點對應。 |

```cpp
/*
// Definition for a Node.
class Node {
public:
    int val;
    Node* next;
    Node* random;
    
    Node(int _val) {
        val = _val;
        next = NULL;
        random = NULL;
    }
};
*/

class Solution {
public:
    Node* copyRandomList(Node* head) {
        unordered_map<Node*,Node*> map;
        Node* current = head;
        if(head == nullptr){
            return nullptr;
        }
        while(current != nullptr){
            map[current] = new Node(current->val);
            current = current->next; 
        }
        map[nullptr] = nullptr;
        current = head;
        while(current != nullptr){
            map[current]->next = map[current->next];
            map[current]->random = map[current->random];
            current = current->next;
        }
        return map[head];
    }
};
```
