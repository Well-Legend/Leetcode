# 56. Merge Intervals

**Link**: https://leetcode.com/problems/merge-intervals/  
**Tags**: Array, Sorting  
**Difficulty**: Medium

## 題目解釋
給定一組區間 `intervals`，其中 `intervals[i] = [start_i, end_i]`。  
請將所有重疊的區間合併，並回傳合併後互不重疊的區間集合。

## 限制與條件
- `1 <= intervals.length <= 10^4`
- `intervals[i].length == 2`
- `0 <= start_i <= end_i <= 10^4`

## 解法思路
### 思考過程
你的程式碼採用經典做法：
1. 先把所有區間依照起點排序。  
2. 準備結果陣列 `ans`。  
3. 依序遍歷排序後的每個區間：
   - 如果 `ans` 是空的，或目前區間和 `ans.back()` 不重疊，就直接加入 `ans`。  
   - 否則代表有重疊，就把 `ans.back()[1]` 更新成兩者終點的較大值，完成合併。  

因為排序後，若有重疊一定只需要和最後一個已合併區間比較，所以邏輯很簡潔。

### 解法：Sort and Merge
> [!IMPORTANT]
> 思路：先依起點排序，再和目前最後一個合併區間比較，決定是延伸還是新開一段。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| 時間複雜度 | O(n log n) | 主要成本來自排序。 |
| 空間複雜度 | O(n) | 需要結果陣列存放合併後區間。 |

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end());
        vector<vector<int>> ans;
        for(int i=0; i<intervals.size(); i++){
            if(ans.empty() || ans.back()[1]<intervals[i][0]){
                ans.push_back(intervals[i]);
            }
            else{
                ans.back()[1] = max(ans.back()[1], intervals[i][1]);
            }
        }
        return ans;
    }
};
```
