# 155. Min Stack

**Link**: https://leetcode.com/problems/min-stack/  
**Tags**: Stack, Design  
**Difficulty**: Medium

## 題目解釋
設計一個支援以下操作的堆疊：
- `push(x)`：將元素推入堆疊
- `pop()`：移除堆疊頂端元素
- `top()`：取得堆疊頂端元素
- `getMin()`：在 `O(1)` 時間內取得目前堆疊中的最小值

## 限制與條件
- `-2^31 <= val <= 2^31 - 1`
- `pop`、`top`、`getMin` 在呼叫時保證堆疊非空
- 各操作需盡量達成 `O(1)`

## 解法思路
### 思考過程
你的程式碼在每個堆疊節點裡，同時存兩個資訊：
1. 實際推入的值 `val`
2. 推入這個位置時，整個堆疊的最小值 `min`  

因此：
- `push(val)` 時，先取得目前最小值，再和新值比較，將新的最小值一起存進去。  
- `pop()` 直接刪掉最後一個元素。  
- `top()` 取最後一個元素的第 0 格。  
- `getMin()` 取最後一個元素的第 1 格。  

這樣每次操作都只碰堆疊尾端，所以可以維持 `O(1)`。

### 解法：Stack with Current Minimum
> [!IMPORTANT]
> 思路：每個位置都額外記錄「到目前為止的最小值」，讓 `getMin()` 直接讀取堆頂資訊。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| `push` | O(1) | 只需比較當前最小值並推入尾端。 |
| `pop` | O(1) | 直接移除尾端元素。 |
| `top` | O(1) | 直接讀取尾端值。 |
| `getMin` | O(1) | 直接讀取尾端記錄的最小值。 |
| 空間複雜度 | O(n) | 每個元素都存一份當前最小值。 |

```cpp
class MinStack {
private:
    vector<vector<int>> stk;
public:
    MinStack() {

    }
    
    void push(int val) {
        int min=getMin();
        if(stk.empty() || val<min){
            min = val;
        }
        stk.push_back({val, min});
    }
    
    void pop() {
        stk.pop_back();
    }
    
    int top() {
        int n=stk.size();
        if(n==0){
            return -1;
        }
        return stk[n-1][0];
    }
    
    int getMin() {
        if(stk.empty()){
            return -1;
        }
        return stk.back()[1];
    }
};

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack* obj = new MinStack();
 * obj->push(val);
 * obj->pop();
 * int param_3 = obj->top();
 * int param_4 = obj->getMin();
 */
```
