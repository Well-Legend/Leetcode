# 146. LRU Cache

**Link**: https://leetcode.com/problems/lru-cache/  
**Tags**: Hash Table, Linked List, Design, Doubly-Linked List  
**Difficulty**: Medium

## 題目解釋
設計並實作一個符合 LRU（Least Recently Used）策略的快取。  
需要支援：
- `get(key)`：若 key 存在，回傳 value，否則回傳 `-1`
- `put(key, value)`：插入或更新 key；若容量超限，移除最久未使用的 key  

兩個操作都必須在平均 `O(1)` 完成。

## 限制與條件
- `1 <= capacity <= 3000`
- `0 <= key <= 10^4`
- `0 <= value <= 10^5`
- 最多呼叫 `2 * 10^5` 次 `get` / `put`

## 解法思路
### 思考過程
你的程式碼使用「hash map + 雙向 linked list」：
1. `unordered_map<int, Node*> map`：用來在 `O(1)` 找到某個 key 對應的節點。  
2. 雙向 linked list：維護使用順序。
   - `head` 後面是最近使用的節點
   - `tail` 前面是最久未使用的節點  

你另外寫了兩個操作：
- `addNode(newNode)`：把節點插到 `head` 後面
- `deleteNode(node)`：把節點從鏈結中移除  

`get(key)`：
- 找不到就回傳 `-1`
- 找到就先移除再插回頭部，表示最近使用  

`put(key, value)`：
- 若 key 已存在，先刪舊節點再移除 map 紀錄
- 若 key 不存在且容量滿了，就刪掉 `tail->prev`，也就是最久未使用的節點
- 最後把新節點插到頭部，並更新 map  

### 解法：Hash Map + Doubly Linked List
> [!IMPORTANT]
> 思路：map 負責 `O(1)` 找節點，雙向 linked list 負責 `O(1)` 更新最近/最久使用順序。

### 複雜度分析
| 維度 | 複雜度 | 說明 |
| --- | --- | --- |
| `get` | O(1) | map 查找加上 linked list 節點移動。 |
| `put` | O(1) | map 更新加上 linked list 插入/刪除。 |
| 空間複雜度 | O(capacity) | 最多存放 capacity 個節點。 |

```cpp
class LRUCache {
private:
    int size;
    struct Node{
        int key, value;
        Node *prev, *next;
        Node(int k, int v): key(k), value(v), prev(nullptr), next(nullptr){}
    };
    unordered_map<int, Node*> map;
    Node *head = new Node(-1,-1);
    Node *tail = new Node(-1,-1);
    
public:
    LRUCache(int capacity) {
        size = capacity;
        head->next = tail;
        tail->prev = head;
    }
    
    void addNode(Node* newNode){
        Node* tmp = head->next;
        newNode->prev = head;
        newNode->next = tmp;
        head->next = newNode;
        tmp->prev = newNode;
    }

    void deleteNode(Node* newNode){
        newNode->prev->next = newNode->next;
        newNode->next->prev = newNode->prev;
    }

    int get(int key) {
        if(map.find(key) == map.end()){
            return -1;
        }
        int val = map[key]->value;
        Node* refreshNode = map[key];
        deleteNode(map[key]);
        addNode(refreshNode);
        map[key] = head->next;

        return val;
    }
    
    void put(int key, int value) {
        Node* newNode = new Node(key, value);
        if(map.find(key) != map.end()){
            deleteNode(map[key]);
            map.erase(key);
        }
        else{
            if(map.size() == size){
                map.erase(tail->prev->key);
                deleteNode(tail->prev);
            }
        }
        addNode(newNode);
        map[newNode->key] = newNode;
    }
};

/**
 * Your LRUCache object will be instantiated and called as such:
 * LRUCache* obj = new LRUCache(capacity);
 * int param_1 = obj->get(key);
 * obj->put(key,value);
 */
```
